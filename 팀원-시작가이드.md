# AI 마케팅 스킬팩 — 팀원 시작 가이드

## 최초 설정 (딱 한 번만)

### 1. Claude Code 설치

터미널 열고 복붙:

```bash
npm install -g @anthropic-ai/claude-code
```

### 2. Claude 로그인

```bash
claude auth login
```

브라우저가 열리면 Claude 계정으로 로그인하세요.

### 3. (선택) meta-ad-factory 서버용 Python 의존성

메타 광고 인터랙티브 편집 기능을 쓰려면:

```bash
pip3 install fastapi 'uvicorn[standard]' beautifulsoup4 python-multipart
```

## 메타 광고 팩토리 — 인터랙티브 편집 워크플로우

```bash
cd skills/advertising/meta-ad-factory/scripts

# 1) 제품 config로 HTML 21개 빌드
python3 products/swaddle_strap.py

# 2) 로컬 서버 시작 → 브라우저 자동 오픈
python3 server.py --slug swaddle-strap
```

브라우저의 preview-grid 에서:
- **📥** 카드별 PNG 단건 다운로드 (on-demand)
- **✏️** 텍스트 + 스타일(색상/크기/굵기/배경) 편집
- **🖼** AI 이미지 교체 (프롬프트 입력 → Gemini 변환)

서버 종료: `Ctrl+C`

## 문제 해결

### "command not found: claude"
→ Node.js 설치 필요: https://nodejs.org

### "인증 실패"
→ 다시 로그인: `claude auth login`

### "스킬을 찾을 수 없음"
→ 올바른 폴더에서 실행했는지 확인:

```bash
cd /Users/inkyo/Projects/team-skills
claude
```

### meta-ad-factory 서버가 "저장 실패" 에러
→ 8765 포트가 다른 프로세스에 점유됨. 확인/해제:

```bash
lsof -i:8765              # 점유 프로세스 확인
lsof -ti:8765 | xargs kill  # 종료
```

## 결과물 저장 위치

모든 결과물은 각 팀원의 로컬 Desktop 하위에 저장됩니다 (Git 레포에는 저장 X):

```
~/Desktop/team-skills/
├── 광고카피/        # 메타 광고, 기타 광고 카피
│   └── sundayhug-meta-bulk/{카테고리}/{제품}/  # meta-ad-factory
├── 영상/            # 영상 소재
├── 리포트/          # 분석 리포트 (경쟁사, 트렌드, 키워드, 신제품 기획 등)
├── 상세페이지/      # 상세페이지 HTML + 섹션 PNG
├── 카드뉴스/        # 카드뉴스 이미지
└── 기타/            # 문서/이미지 변환 결과 등
```

> 해당 폴더는 스킬이 실행되며 필요할 때 자동 생성됩니다 (`mkdir -p`).
