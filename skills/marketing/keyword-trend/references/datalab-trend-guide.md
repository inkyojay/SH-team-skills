# DataLab 트렌드 분석 상세 가이드

## MCP 도구 사용법

### 기본 검색 트렌드 (datalab_search)

가장 핵심적인 도구. 키워드의 월별 검색량 상대 추이를 제공한다.

#### 파라미터 구성

```json
{
  "startDate": "2024-03-01",
  "endDate": "today",
  "timeUnit": "month",
  "keywordGroups": [
    {
      "groupName": "메인키워드",
      "keywords": ["키워드1", "키워드1변형", "키워드1유사어"]
    },
    {
      "groupName": "비교키워드",
      "keywords": ["키워드2", "키워드2변형"]
    }
  ]
}
```

#### 핵심 규칙
- `keywordGroups`는 최대 **5개**
- 그룹당 `keywords`는 최대 **20개**
- 같은 그룹 내 키워드는 합산되어 하나의 트렌드로 표시
- `ratio`는 전체 기간 중 최고점을 100으로 환산한 상대값
- `timeUnit`은 "date"(일별), "week"(주별), "month"(월별) 선택 가능

#### 분석 기간 가이드
- 시즌 패턴 파악: 최소 2년 (24개월)
- 단기 트렌드: 6개월
- 장기 시장 변화: 3~5년

### 쇼핑 카테고리 트렌드 (datalab_shopping_category)

네이버 쇼핑에서의 카테고리별 클릭 트렌드. 실제 구매 의도를 반영한다.

#### 카테고리 코드 찾기
`find_category` 도구로 먼저 카테고리 코드를 조회해야 한다.
```
find_category("슬리핑백") -> 카테고리 코드 확인
```

#### 파라미터
```json
{
  "startDate": "2024-03-01",
  "endDate": "today",
  "timeUnit": "month",
  "category": [
    {"name": "아기수면용품", "param": ["카테고리코드"]},
    {"name": "아기의류", "param": ["카테고리코드"]}
  ]
}
```

### 쇼핑 키워드 비교 (datalab_shopping_keywords)

같은 카테고리 내에서 키워드 간 클릭 트렌드 비교.

```json
{
  "startDate": "2024-03-01",
  "endDate": "today",
  "timeUnit": "month",
  "category": "카테고리코드",
  "keyword": [
    {"name": "슬리핑백", "param": ["슬리핑백"]},
    {"name": "수면조끼", "param": ["수면조끼"]}
  ]
}
```

## 시즌 패턴 분석 방법

### 1. 피크/비수기 자동 판별

2년치 데이터에서 같은 월의 ratio를 평균내어 시즌 패턴을 도출한다.

```
월별 평균 = (작년 N월 ratio + 올해 N월 ratio) / 2
피크월 = 평균이 가장 높은 월
비수기 = 평균이 가장 낮은 월
시즌 강도 = 피크월 평균 / 비수기 평균
```

### 2. 시즌 유형별 마케팅 캘린더

#### 강한 계절성 (시즌 강도 > 3배)
```
마케팅 준비: 피크 -2개월 (콘텐츠 제작, SEO 최적화)
광고 시작:   피크 -1개월 (예산 30% 투입)
피크 집중:   피크월 (예산 50% 투입)
피크 유지:   피크 +1개월 (예산 20% 투입)
비수기:      나머지 (SEO 유지, 콘텐츠 축적)
```

#### 약한 계절성 (시즌 강도 1.5~3배)
```
피크 시즌: 균등 배분 대비 +30% 예산
비수기:    균등 배분 대비 -20% 예산
```

#### 연중 안정 (시즌 강도 < 1.5배)
```
연간 균등 배분, 출산 시즌(3~4월, 9~10월) 소폭 증액
```

### 3. YoY 성장/하락 판단

```
YoY 변화 = (올해 ratio - 작년 ratio) / 작년 ratio * 100

+20% 이상: 급성장 (시장 확대 중, 선점 기회)
+5~20%:    성장 (긍정적)
-5~+5%:    유지 (안정적 시장)
-5~-20%:   하락 (경쟁 심화 또는 시장 축소)
-20% 이하: 급하락 (카테고리 전환 고려)
```

## 실전 활용 예시

### 아기 수면용품 분석 (썬데이허그 기준)

검색량 + 트렌드 결합 결과:

| 키워드 | 월검색량 | 시즌유형 | 피크 | 비수기 | 강도 | 전략 |
|--------|---------|---------|------|-------|------|------|
| 수면조끼 | 중 | 강한 계절성 | 10월 | 6월 | 6.4배 | 8월 준비, 9~11월 집중 |
| 슬리핑백 | 중 | 약한 계절성 | 10월 | 6월 | 1.8배 | 9~10월 소폭 증액 |
| 속싸개 | 대 | 연중 안정 | 8월 | 4월 | 1.4배 | 상시 마케팅 |
| 스와들업 | 대 | 연중 안정 | 10월 | 6월 | 1.3배 | 브랜드 경쟁 (상시) |
| 아기이불 | 중 | 강한 계절성 | 11월 | 7월 | 4.0배 | 9월 준비, 10~12월 집중 |

### 마케팅 캘린더 도출

```
1~3월:  속싸개/스와들업 상시 마케팅 유지
4~6월:  비수기 - SEO 콘텐츠 축적, 신제품 기획
7월:    수면조끼/아기이불 콘텐츠 제작 시작
8월:    수면조끼/아기이불/슬리핑백 광고 소재 준비
9월:    가을 시즌 광고 개시 (전 제품)
10월:   수면조끼/슬리핑백 피크 - 최대 예산
11월:   아기이불 피크 - 예산 유지
12월:   하강기 - 예산 축소, 성과 분석
```
