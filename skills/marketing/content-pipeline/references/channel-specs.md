# 채널별 콘텐츠 규격

## 네이버 블로그

| 항목 | 규격 |
|------|------|
| 제목 | 최대 100자 (검색 노출 40자 권장) |
| 본문 | 권장 1,500~3,000자 |
| 이미지 | 본문 내 3~10장, 대표이미지 1장 |
| 키워드 밀도 | 1.5~2.5% (자연스럽게) |
| 태그 | 최대 10개 |
| 발행 시간 | 오전 8-10시, 저녁 8-10시 최적 |

### SEO 체크리스트
- [ ] 제목에 핵심 키워드 포함
- [ ] 첫 문단에 키워드 1회 이상
- [ ] H2/H3 소제목 3개 이상
- [ ] 이미지 ALT 텍스트에 키워드
- [ ] 내부 링크 (이전 포스트) 1~2개
- [ ] 외부 링크 (출처) 1개

## 인스타그램 피드

| 항목 | 규격 |
|------|------|
| 캡션 | 최대 2,200자 (첫 125자가 미리보기) |
| 해시태그 | 15~20개 (대중5 + 니치10 + 브랜드5) |
| 이미지 | 1:1 (1080×1080) 또는 4:5 (1080×1350) |
| 캐러셀 | 최대 10장, 권장 4~8장 |
| 발행 시간 | 오전 7-9시, 점심 12-1시, 저녁 7-9시 |

### 해시태그 전략
```
대중 (50만+): #육아, #맘스타그램, #아기용품, #육아템, #임산부
니치 (1~50만): #아기슬리핑백, #아기수면, #꿀잠템, #신생아용품
브랜드: #썬데이허그, #sundayhug, #아기꿀잠
시즌: #여름육아, #열대야아기, #아기여름용품
```

## 인스타그램 릴스

| 항목 | 규격 |
|------|------|
| 길이 | 15~30초 (최대 90초) |
| 비율 | 9:16 (1080×1920) |
| 캡션 | 피드와 동일 (2,200자) |
| 커버 이미지 | 1080×1920 |
| 자막 | 필수 (무음 시청 80%) |
| 발행 시간 | 오전 9시, 오후 12시, 저녁 7시 |

### 릴스 구조 템플릿
```
[0-3초]  HOOK — 질문/충격/비포 (시청 유지의 핵심)
[3-10초] MAIN — 핵심 메시지/데모
[10-25초] DETAIL — 상세 설명/비교
[25-30초] CTA — "저장해두세요" / "프로필 링크"
```

## 스레드 (Threads)

| 항목 | 규격 |
|------|------|
| 텍스트 | 최대 500자 |
| 이미지 | 최대 10장 |
| 해시태그 | 3~5개 (과도하면 역효과) |
| 톤 | 캐주얼, 대화형, 진정성 |
| 발행 시간 | 점심/저녁 |

### 스레드 포맷 패턴
```
1. 질문형: "요즘 아기 이불 어떻게 하세요?"
2. 고백형: "솔직히 이거 쓰기 전까지 잠을 못 잤어요"
3. 리스트형: "아기 꿀잠 3가지 비밀"
4. 논쟁형: "모로반사 속싸개 언제까지 써야 할까요?"
```

## 카카오 알림톡

| 항목 | 규격 |
|------|------|
| 제목 | 최대 20자 |
| 본문 | 최대 1,000자 (권장 50~100자) |
| 버튼 | 최대 5개 (웹링크/앱링크) |
| 이미지 | 와이드형 800×400 |
| 발송 시간 | 오전 10시, 오후 2시 최적 |

### 알림톡 카피 공식
```
[제목] 혜택/가치 중심 (20자)
[본문] 왜 지금 봐야 하는지 + CTA (50~100자)
[버튼] 액션 유도 ("지금 확인하기", "할인 받기")
```

## 카드뉴스

| 항목 | 규격 |
|------|------|
| 장 수 | 4~8장 (권장 6장) |
| 크기 | 1:1 (1080×1080) 또는 4:5 (1080×1350) |
| 텍스트 | 장당 3줄 이내 |
| 표지 | 후크 질문 또는 숫자 ("5가지 비밀") |
| 마지막 장 | CTA + 브랜드 로고 |

### 카드뉴스 구조
```
1장: 후크 (질문/숫자/충격)
2~5장: 핵심 콘텐츠 (1장 1메시지)
6장: CTA + 저장 유도 + 브랜드
```
