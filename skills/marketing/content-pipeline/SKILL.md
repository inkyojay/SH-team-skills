# Content Pipeline — 트렌드 기반 콘텐츠 자동 기획·생산

## 개요

```
Trend Radar 데이터       →  Content Planner        →  Content Factory        →  Distribution
(급상승 키워드 +            (키워드×포맷×채널           (채널별 콘텐츠              (저장 + 알림)
 트렌딩 사운드 +             매트릭스 자동 생성)          벌크 생성)
 인기 포맷)
```

## 파이프라인 상세

### STEP 1: 인풋 수집

**자동 모드** (Trend Radar 연동):
- trend-radar의 위클리 리포트 JSON에서 자동으로 인풋 추출
- 급상승 키워드 TOP 5 + 트렌딩 사운드 TOP 3 + 인기 포맷 TOP 3

**수동 모드**:
- 사용자가 직접 키워드/주제 입력
- "아기 쿨매트로 이번 주 콘텐츠 만들어줘"

### STEP 2: Content Planner (기획 매트릭스)

트렌드 인사이트에서 콘텐츠 기획안을 자동 생성한다.

**매트릭스 구조: 키워드 × 포맷 × 채널**

| 키워드 | 포맷 | 블로그 | 인스타 피드 | 릴스 | 스레드 | 카카오 |
|--------|------|--------|------------|------|--------|--------|
| 아기 쿨매트 | 비교 리뷰 | O | O (캐러셀) | - | O | - |
| 아기 쿨매트 | Before/After | - | - | O | - | - |
| 여름 슬리핑백 | 가이드 | O | O | O | O | O |

**우선순위 결정 기준:**
1. 검색량 증가율 (높을수록 우선)
2. 경쟁도 (낮을수록 우선)
3. 기존 콘텐츠 유무 (신규 키워드 우선)
4. 시즌 적합성 (현재 시즌에 맞는 것 우선)
5. 상품 연관성 (썬데이허그 제품과 직접 연관 우선)

매트릭스를 표로 보여주고 사용자 확인 후 STEP 3으로 진행.

### STEP 3: Content Factory (채널별 벌크 생성)

하나의 키워드/주제에서 여러 채널 콘텐츠를 한 번에 생성한다.

#### 네이버 블로그 (SEO 최적화)
→ **기존 naver-blog-seo-writer 스킬 연동**

생성 항목:
- SEO 타이틀 (검색 노출 최적화)
- 본문 2,000자 내외 (H2/H3 구조)
- 키워드 밀도 1.5~2.5%
- Gemini API 이미지 3~5장
- 메타 디스크립션

#### 인스타그램 피드 (캐러셀/싱글)
생성 항목:
- 캡션 (2,200자 이내, 처음 125자가 핵심)
- 해시태그 세트 (15~20개, 대중/니치/브랜드 믹스)
- 캐러셀 슬라이드 구성 (4~10장)
- 첫 장 후크 텍스트

#### 인스타그램 릴스
생성 항목:
- 릴스 스크립트 (15~30초)
- 장면별 비주얼 디렉션
- 후크 (처음 3초) 
- CTA
- 추천 트렌딩 사운드 (Trend Radar 데이터 자동 매칭)
- 캡션 + 해시태그

#### 스레드 (Threads)
생성 항목:
- 짧은 텍스트 (500자 이내)
- 캐주얼 맘 톤
- 대화 유도형 질문 포함
- 연관 해시태그 3~5개

#### 카카오 알림톡
생성 항목:
- 푸시 제목 (20자 이내)
- 푸시 본문 (50자 이내)
- 랜딩 URL
- 발송 타이밍 추천

#### 카드뉴스
→ **기존 card-news 스킬 연동**

생성 항목:
- 4~8장 구성
- 슬라이드별 헤드라인 + 본문
- 비주얼 디렉션

### STEP 4: 주간 콘텐츠 캘린더

트렌드 데이터 + 시즌 이벤트 + 프로모션 캘린더를 종합하여 주간 발행 스케줄 생성.

```
| 날짜 | 채널 | 콘텐츠 유형 | 주제 | 상태 |
|------|------|------------|------|------|
| 월 | 블로그 | SEO 가이드 | 여름 슬리핑백 비교 | 작성 완료 |
| 화 | 인스타 피드 | 캐러셀 | 쿨매트 Before/After | 작성 완료 |
| 수 | 릴스 | 15초 숏폼 | 꿀잠 팁 | 스크립트 완료 |
| 목 | 스레드 | 대화형 | 여름 수면 고민 | 작성 완료 |
| 금 | 카카오 | 알림톡 | 쿨매트 신상 안내 | 작성 완료 |
```

### STEP 5: 성과 피드백 루프 (선택)

Instagram Graph API로 게시물 성과 수집:
- 도달, 저장, 공유, 댓글 수
- 잘 된 포맷/소구점 학습
- 다음 주 콘텐츠에 반영

**"잘 되는 걸 더 만드는" 자동 루프.**

### 파일 저장
각 채널 콘텐츠는 별도 파일로 저장:
```
output/콘텐츠/
├── 2026-W15/
│   ├── plan.json                          # 주간 기획안
│   ├── calendar.md                        # 주간 캘린더
│   ├── blog-아기쿨매트비교.md              # 블로그 원고
│   ├── instagram-아기쿨매트-caption.md     # 인스타 캡션
│   ├── reels-아기쿨매트-script.md          # 릴스 스크립트
│   ├── threads-아기쿨매트.md               # 스레드
│   └── kakao-아기쿨매트.md                 # 카카오 알림톡
```

## 기존 스킬 연동

| 기존 스킬 | 연동 방식 |
|----------|----------|
| **trend-radar** | 트렌드 데이터 입력 (급상승 키워드, 트렌딩 사운드, 인기 포맷) |
| **naver-blog-seo-writer** | 블로그 콘텐츠 생성 시 SEO 최적화 로직 활용 |
| **meta-ad-factory** | 유료 프로모션 소재 필요 시 광고 크리에이티브 생성 연동 |
| **card-news** | 카드뉴스 콘텐츠 생성 시 카드뉴스 스킬 연동 |
| **promotion-planner** | 프로모션 캘린더와 콘텐츠 캘린더 동기화 |
| **sundayhug-marketing-planner** | 전체 마케팅 전략과 정합성 확인 |

## 톤앤매너 가이드

### 블로그 (네이버 SEO)
- 정보성 전문가 톤
- "~입니다" 체
- 키워드 자연스럽게 배치
- 구매 결정에 도움되는 비교/분석 중심

### 인스타그램
- 따뜻하고 공감 가는 톤
- 이모지 적절히 사용
- 첫 줄에 후크 (질문 or 공감)
- CTA: "저장해두세요", "링크 확인"

### 릴스
- 처음 3초 후크 필수
- 짧고 임팩트 있는 문장
- 트렌딩 사운드 활용
- 자막 텍스트 오버레이

### 스레드
- 캐주얼 맘 톤
- "~요" 체
- 대화 유도형 질문
- 솔직하고 공감 가는 톤

### 카카오 알림톡
- 극도로 짧고 명확
- 혜택/가치 중심
- 긴급성 또는 호기심 유발
