# 소싱 가이드 (1688 / Alibaba)

## 1688.com (중국 내수 도매)

### 검색 방법

**방법 A: web_search**
```
web_search: "site:1688.com {중문 키워드}"
예: "site:1688.com 竹纤维婴儿睡袋" (대나무 아기 슬리핑백)
```

**방법 B: Apify Actor**
```
Actor: apify/1688-scraper
Input: { keyword: "婴儿睡袋", maxItems: 50 }
```

### 수집 데이터

| 항목 | 용도 |
|------|------|
| 가격 범위 | 원가 추정 (× 1.3 관세/물류 포함) |
| MOQ | 최소 주문 수량 → 초기 투자 규모 |
| 공급자 등급 | 실력상/알리바바 인증 여부 |
| 거래 건수 | 공급자 신뢰도 |
| OEM/ODM 여부 | 커스터마이징 가능성 |
| 인증 | CCC, CE, FDA 등 |
| 샘플 가격 | 테스트 비용 |

### 원가 계산 공식

```
예상 총원가 = 1688 가격 + 관세(8~13%) + 물류비 + 검수비
          ≈ 1688 가격 × 1.3

예상 마진율 = (판매가 - 총원가) / 판매가 × 100

예시:
  1688 가격: 15 CNY (≈ 2,800원)
  총원가: 2,800 × 1.3 = 3,640원
  판매가: 39,000원
  마진율: (39,000 - 3,640) / 39,000 = 90.7% (제조원가 기준)
  
실제 마진 (마케팅/운영 포함):
  총원가 + 마케팅(15%) + 수수료(10%) + 물류(5%) + 운영(5%)
  = 3,640 + 5,850 + 3,900 + 1,950 + 1,950 = 17,290원
  실질 마진: (39,000 - 17,290) / 39,000 = 55.7%
```

## Alibaba.com (글로벌 도매)

### 검색 방법

**방법 A: web_search**
```
web_search: "site:alibaba.com {영문 키워드}"
예: "site:alibaba.com bamboo baby sleep sack OEM"
```

**방법 B: Apify Actor**
```
Actor: apify/alibaba-scraper
Input: { keyword: "bamboo baby sleep sack", maxItems: 50 }
```

### 1688 vs Alibaba 비교

| 항목 | 1688 | Alibaba |
|------|------|---------|
| 시장 | 중국 내수 | 글로벌 |
| 언어 | 중문 | 영문 |
| 가격 | 더 저렴 | 약간 높음 |
| MOQ | 보통 더 낮음 | 보통 더 높음 |
| 커뮤니케이션 | 중국어 필요 | 영어 가능 |
| 샘플 | 가능 | 가능 |
| 결제 | 알리페이 | 다양 |
| 추천 용도 | 가격 우선 | 커뮤니케이션 편의 |

## Amazon BSR (해외 트렌드 감지)

### 검색 방법

```
web_search: "amazon best sellers baby sleep" site:amazon.com
web_search: "amazon new releases baby nursery" site:amazon.com
```

### 분석 포인트

| 항목 | 감지 대상 |
|------|----------|
| New Releases | 신규 진입 제품 → 트렌드 초기 신호 |
| Movers & Shakers | BSR 급상승 제품 → 수요 급등 |
| 리뷰 키워드 | 소비자 니즈/불만 → 제품 개선 기회 |
| 가격대 | 해외 가격 → 국내 가격 전략 참고 |
| 한국 미진출 | 국내에 없는 제품 → 블루오션 |
