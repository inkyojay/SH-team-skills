# 단일 경쟁사 심층분석 리포트 스펙

## 리포트 구조 (탭 7개)

### TAB 1: 📊 프로필 요약 (Brand Profile)
- 브랜드명, 로고(이미지 가능시), 설립연도
- 채널 맵: 네이버 스토어 / 쿠팡 / 자사몰 / 인스타 (링크 버튼)
- 핵심 수치 카드 4개:
  - 총 상품 수
  - 평균 가격대
  - 총 리뷰 수 (근사치)
  - 인스타 팔로워 (근사치)
- 한 줄 요약: AI가 분석한 브랜드 포지셔닝 문장

### TAB 2: 🛒 제품 & 가격 (Products & Pricing)
- 전체 상품 테이블 (검색/정렬 가능):
  - 상품명 | 정가 | 할인가 | 할인율 | 리뷰수 | 평점 | 카테고리
- 베스트셀러 TOP 10 카드
- 가격 분포 히스토그램 (Chart.js bar)
- 카테고리별 평균 가격 비교
- 가격 전략 인사이트 (무배/묶음할인/쿠폰 등)

### TAB 3: 🏷️ 프로모션 (Promotions)
- 현재 진행 중인 프로모션 카드 리스트
  - 프로모션명 | 유형 | 기간 | 할인율 | 대상 상품
- 체험단/인플루언서 활동 타임라인
- 최근 3개월 프로모션 이력 (확인 가능한 경우)
- 프로모션 유형 분포 파이 차트

### TAB 4: 📱 마케팅 분석 (Marketing)
- 인스타그램 섹션:
  - 게시 빈도 (일평균)
  - 콘텐츠 유형 분포 도넛 차트
  - 주요 해시태그 버블 차트 (또는 태그 클라우드)
  - 최근 인기 게시물 TOP 5 (캡션 요약)
  - 협찬/광고 비율
- 네이버 블로그 섹션:
  - 체험단 추정 규모 (최근 1개월 블로그 수)
  - 주요 블로그 키워드
- 검색 마케팅:
  - 파워링크/브랜드검색 집행 여부
  - 관련 키워드 트렌드

### TAB 5: ⭐ 리뷰 & VOC (Reviews)
- 전체 감성 분포 도넛 차트 (긍정/부정/중립)
- 장점 키워드 TOP 10 (바 차트, 녹색)
- 불만 키워드 TOP 10 (바 차트, 빨강)
- 대표 리뷰 발췌 (긍정 3건 / 부정 3건)
  - 리뷰 텍스트 + 별점 + 날짜
- 맘카페/블로그 VOC 요약

### TAB 6: ⚔️ vs 썬데이허그 비교 (Comparison)
- 비교 매트릭스 테이블:
  | 항목 | 경쟁사 | 썬데이허그 | 판정 |
  |------|--------|-----------|------|
  | 가격대 | | | |
  | 상품 수 | | | |
  | 평균 리뷰수 | | | |
  | 인스타 팔로워 | | | |
  | 마케팅 강도 | | | |
- 가격대 오버랩 차트 (두 브랜드 가격 분포 겹침)
- 강점/약점 SWOT 2x2 비교
- 제품 카테고리별 직접 대응 제품 매핑

### TAB 7: 💡 인사이트 & 액션 (Insights)
- **배울 점** (경쟁사의 강점 중 벤치마크할 것) — 3~5건
- **우리의 기회** (경쟁사 약점에서 발견한 기회) — 3~5건
- **즉시 대응 필요** (긴급도 높은 액션) — 0~3건
  - 🔴 긴급 / 🟡 권장 / 🟢 검토 라벨
- **추가 조사 필요 항목** (데이터 부족으로 추가 확인 필요)
