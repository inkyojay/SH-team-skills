## 구현 옵션

### Option A: Supabase Edge Function (권장)
- **장점**: 서버리스, Mac 상시 켜짐 불필요
- **실행**: Supabase Cron (`pg_cron`) + Edge Function
- **적합**: 안정적 매일 자동화

```typescript
// supabase/functions/ci-daily-run/index.ts
Deno.cron("CI Daily Run", "0 22 * * *", async () => {  // UTC 22:00 = KST 07:00
  // 1. Load competitors from ci_competitors table
  // 2. Run price monitoring (web_fetch 기반)
  // 3. Run social snapshots
  // 4. Run social listening
  // 5. Generate HTML report → Supabase Storage
  // 6. Send Slack alert if changes detected
  // 7. Monday: Send weekly email summary
});
```

### Option B: Mac Mini Cron (개발/테스트용)
```bash
# crontab -e
0 7 * * * /usr/local/bin/python3 /path/to/ci_runner.py >> /logs/ci.log 2>&1
```

### Option C: Claude Code Skill 직접 실행 (수동)
- Claude Code에서 `competitive-intelligence` 스킬 호출
- 결과 즉시 확인
- 리포트 다운로드

## 알림 중복 방지 규칙

- 동일 브랜드 + 동일 변동 유형: 24시간 내 중복 알림 금지
- 가격 변동은 ±10% 이상만 알림 (소규모 변동 무시)
- 슬랙 알림 최대 5건/일 (초과 시 하나로 묶어서 발송)
