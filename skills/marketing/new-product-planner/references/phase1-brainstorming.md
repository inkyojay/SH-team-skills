# PHASE 1 — 아이디어 브레인스토밍 상세 가이드

## 진행 원칙

이 단계는 **기획 회의**다. Claude는 회의 진행자이자 공동 기획자 역할을 한다.
절대 일방적으로 아이디어를 쏟아내지 않는다. 사용자의 생각을 먼저 듣고, 그 위에 살을 붙인다.

## 시작 — 씨앗 질문

사용자의 상태에 따라 첫 질문을 달리한다:

### 상태 A: 막연한 상태 ("뭘 만들면 좋을까?")
질문 순서:
1. "요즘 고객들한테서 가장 많이 듣는 불만이나 요청이 뭐가 있으세요?"
2. "우리 기존 제품 중에서 '이거 좀 더 이러면 좋겠는데' 싶은 게 있나요?"
3. "최근에 경쟁사나 해외에서 본 제품 중 눈에 띄는 게 있었나요?"

### 상태 B: 방향은 있는 상태 ("OO 쪽으로 생각 중이야")
질문 순서:
1. "그 아이디어가 떠오르게 된 계기가 있으세요? (고객 반응? 트렌드?)"
2. "구체적으로 어떤 형태의 제품을 상상하고 계세요?"
3. "타겟 고객은 기존과 같을까요, 새로운 고객층인가요?"

### 상태 C: 구체적인 아이디어 ("이런 제품 만들려고 해")
질문 순서:
1. "그 제품의 가장 큰 강점이 뭐라고 보세요?"
2. "기존에 비슷한 제품이 시장에 있나요? 있다면 어떤 점이 다를까요?"
3. "언제쯤 출시를 생각하고 계세요?"

## 확장 — 아이디어 발전

사용자의 답변을 듣고 나서:

1. **연관 아이디어 제안**: 사용자 아이디어에서 파생될 수 있는 변형 2~3개 제안
2. **트렌드 연결**: 최근 소비 트렌드와 연결 가능한 포인트 언급
3. **타겟 확장**: 기존 타겟 외에 노릴 수 있는 새로운 고객층 제안
4. **시즌/타이밍**: 출시 시기에 따른 기회/리스크 언급

## 정리 — 아이디어 카드화

충분히 대화한 후 (최소 3~4번 주고받기), 나온 아이디어들을 카드 형태로 정리:

```
## 아이디어 카드 #1: [가칭]
- 컨셉: [한 줄 설명]
- 타겟: [주요 고객]
- 해결 문제: [페인포인트]
- 차별화: [기존 대비 다른 점]
- 매력도: ★★★★☆ (Claude 평가)
- 시즌: [최적 시기]
```

3~5개 카드를 정리한 후, 사용자에게 선택을 요청:
"이 중에서 가장 끌리는 아이디어가 어떤 건가요? 1~2개 골라주시면 다음 단계로 깊이 파볼게요."

## 썬데이허그 특화 — 아이디어 방향 힌트

썬데이허그 모드에서는 다음 방향을 자연스럽게 제안할 수 있다:
- 기존 슬리핑백/속싸개 라인의 **시즌 확장** (여름 쿨링 / 겨울 보온 특화)
- **연령 확장**: 0~12개월 중심 → 유아기(2~5세)까지
- **카테고리 확장**: 수면 → 외출 / 목욕 / 놀이 등 인접 카테고리
- **소재 혁신**: 새로운 기능성 소재 (항균, 온도 조절, 오가닉 등)
- **부모 제품**: 아기용이 아닌 부모를 위한 보조 제품
- **세트/번들**: 기존 제품 조합의 새로운 패키지
