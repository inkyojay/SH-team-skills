# PHASE 4 — 롱테일 키워드 발굴 상세 가이드

## 키워드 발굴 프로세스

### 소스 1: 제품 속성 조합 키워드 (10개+)

제품의 속성을 조합해 키워드를 생성한다:

조합 공식: [타겟] + [소재/특성] + [제품 카테고리] + [용도/시즌]

예시 (슬리핑백 기준):
- 신생아 + 메쉬 + 슬리핑백 + 여름용
- 돌아기 + 오가닉코튼 + 슬리핑백 + 사계절
- 뒤집기 + 아기 + 수면조끼 + 겨울

### 소스 2: 고객 언어 기반 키워드 (10개+)

실제 고객이 사용하는 자연어 검색 패턴 수집:

web_search 쿼리 패턴:
- "네이버 지식인 [카테고리] 추천"
- "맘카페 [카테고리] 질문"
- "[카테고리] 어떤 거 사요"
- "[관련 고민] 해결 방법"

수집 포인트:
- 질문 제목에 사용된 표현 그대로 → 키워드로 활용
- "~하는 이유", "~언제부터", "~추천" 패턴 주목
- 브랜드명 없이 카테고리/고민으로 검색하는 패턴

### 소스 3: 경쟁사 키워드 역추적 (5개+)

PHASE 3에서 조사한 경쟁사 정보 활용:
- 경쟁사 상세페이지 제목/키워드
- 경쟁사 블로그 포스팅 제목
- 경쟁사가 광고하고 있는 키워드

### 소스 4: 연관 검색어 확장 (5개+)

web_search로 시드 키워드 검색 후:
- 네이버 자동완성 키워드 확인
- "연관 검색어" 섹션 확인
- "사람들이 많이 본" 키워드 확인

## 키워드 분류 & 정리

발굴한 키워드를 아래 형식으로 정리:

| # | 키워드 | 검색의도 | 예상경쟁도 | 추천채널 | 우선순위 |
|---|--------|---------|-----------|---------|---------|
| 1 | 아기 이불 걷어차요 | 문제해결 | 낮음 | 블로그+맘카페 | ★★★ |
| 2 | 메쉬 슬리핑백 추천 | 구매의도 | 중간 | 배포형+스마트스토어 | ★★★ |

## 우선순위 판단 기준

| 구분 | 조건 | 공략 전략 |
|------|------|---------|
| 1순위 골든 | 검색의도 높음 + 경쟁 낮음 | 즉시 콘텐츠 제작 |
| 2순위 성장 | 검색의도 중간 + 트렌드 상승 | SEO 선점 |
| 3순위 보조 | 검색량 있으나 경쟁 높음 | 참고/보조 키워드 |

## PHASE 5 연결

이 단계에서 발굴한 키워드 리스트가 PHASE 5 네이버 API 조사의 입력값이 된다.
최소 30개, 이상적으로 50개 이상의 키워드를 준비하여 PHASE 5로 넘긴다.
