# PHASE 9 — 런칭 마케팅 전략 & 액션플랜 가이드

## 목적

PHASE 1~8의 모든 결과물(제품 컨셉, 시장 데이터, VOC, 경쟁사, 키워드, 가격, 소구점)을 바탕으로 실제 런칭에 필요한 마케팅 전략과 주차별 실행 계획을 수립한다.

이 단계는 기존 썬데이허그 마케팅 스킬들의 프레임워크를 참고하되, 신제품 런칭에 특화된 전략으로 재구성한다:
- `sundayhug-marketing-planner` → 채널별 콘텐츠 전략, 키워드 우선순위
- `promotion-planner` → 프로모션 유형, 주차별 액션
- `marketing-content-factory` → 콘텐츠 템플릿, 채널별 가이드

## 런칭 전략 3단계 프레임워크

### STAGE 1: 사전 준비 (런칭 D-30 ~ D-7)

**목표**: 검색 기반 구축 + 인지도 씨 뿌리기

| 채널 | 액션 | 수량/빈도 | 목적 |
|------|------|---------|------|
| 스마트스토어 | 상품 등록 + 키워드 최적화 | 전 제품 | 검색 노출 기반 |
| 네이버 블로그 | 정보형 콘텐츠 선행 발행 | 5~8편 | SEO 선점 |
| 배포형 블로그 | 문제해결형 키워드 공략 | 3~5건 | 상위 노출 |
| 인스타그램 | 티징 콘텐츠 (곧 만나요) | 3~5개 | 기대감 형성 |
| 맘카페 | 관련 질문글에 자연스러운 정보 제공 | 간접 | 사전 인지 |

콘텐츠 전략 (PHASE 4~5 키워드 활용):
- **1순위**: 문제해결형 키워드 → 블로그 (PHASE 4에서 발굴한 골든 키워드)
  - 예: "아기 여름밤 자주 깨요 — 원인과 해결법"
  - 자연스럽게 제품 소개 연결
- **2순위**: 정보탐색형 키워드 → 블로그 시리즈
  - 예: "아기 적정 수면 온도 여름철 가이드"
- 상세페이지 SEO: PHASE 4 구매의도형 키워드를 제목/태그에 반영

### STAGE 2: 런칭 폭격 (D-Day ~ D+30)

**목표**: 초기 판매량 + 리뷰 확보 + 바이럴 점화

| 주차 | 핵심 테마 | 주요 액션 |
|------|---------|---------|
| 1주차 | 런칭 특가 | 얼리버드 할인 오픈 + 인스타 런칭 릴스 + 기존 고객 DM/알림 |
| 2주차 | 체험단 폭발 | 체험단 20명 모집 + 맘카페 체험단 공지 + 블로그 리뷰 배포 시작 |
| 3주차 | 비교 콘텐츠 | "A vs B" 블로그 집중 + 배포형 비교 원고 5건 + 인스타 카드뉴스 |
| 4주차 | 리뷰 수확 | 포토리뷰 캠페인 + 리뷰 이벤트 + 맘카페 자연 후기 확산 |

프로모션 전략 (PHASE 7 가격 전략 연계):
- 1주차: 얼리버드 15~20% 할인 (사전예약 고객 or 기존 고객)
- 2~3주차: 런칭 특가 10~15% (신규 고객 유입)
- 4주차: 세트 구매 혜택 (객단가 UP)

### STAGE 3: 성장 가속 (D+31 ~ D+90)

**목표**: 검색 점유율 확대 + 바이럴 자생 + 수익 안정화

| 주차 | 핵심 테마 | 주요 액션 |
|------|---------|---------|
| 5~6주차 | 바이럴 확산 | 맘카페 공동구매 1차 + UGC 해시태그 캠페인 + 인스타 릴스 주 3회 |
| 7~8주차 | 검색 강화 | 배포형 집중 (구매의도 키워드) + 블로그 시리즈 완결 |
| 9~10주차 | 세트/업셀 | 쿨슬립 세트 프로모션 + 크로스셀 (슬리핑백 고객 → 쿨매트) |
| 11~12주차 | 수확+전환 | 정상가 전환 + 시즌 마감 세일 예고 + 성과 분석 리포트 |

## 채널별 상세 전략

### 네이버 블로그/스마트스토어 — SEO 자산 구축
기존 `sundayhug-marketing-planner` STEP 5의 프레임워크 적용:
- 브랜드 발행: 월 4회 정보형 롱폼 (PHASE 4 키워드 기반)
- 체험단 UGC: 월 8~12건 (PHASE 8 소구점 가이드 제공)
- 스마트스토어: 상품명에 골든 키워드 반영, 포토리뷰 유도

### 인스타그램 — 감성 브랜딩 + 바이럴
- 피드: 주 3회 (제품 라이프스타일 컷 + 아기 수면 팁)
- 릴스: 주 2회 (정보형 숏폼 — "여름 아기 꿀잠 팁 3가지")
- 스토리: 프로모션 알림 + 사용자 후기 리포스트
- UGC 캠페인: 해시태그 이벤트 (#썬데이허그쿨슬립 #아기여름꿀잠)

### 맘카페 — 진성 바이럴 (핵심 전환 채널)
기존 `promotion-planner`의 맘카페 전략 적용:
- 체험단 진성 후기 자연 확산
- 정보글 바이럴 (PHASE 2.6 VOC에서 나온 질문에 답하는 형태)
- 공동구매 제휴 (리뷰 50건+ 확보 후)
- **절대 원칙: 브랜드 계정 직접 참여 금지**

### 배포형 블로그 — 검색 상위 노출
- PHASE 4 비교탐색형 키워드 → 비교 콘텐츠 집중
- 파워블로거 원고 의뢰 (건당 8~15만원)
- 월 6~10건 목표

### 스레드 — 커뮤니티 신뢰 구축
- 육아 꿀팁 주 3~4회 (광고 느낌 없이)
- 투표/질문형 소통으로 팔로워 확보
- 자연스러운 제품 언급 (비율: 정보 80% + 제품 20%)

## 콘텐츠 소재 자동 생성

PHASE 8 소구점을 `marketing-content-factory` 형식으로 변환:

각 소구점마다:
- **블로그 제목 3개** (SEO 최적화 + 클릭 유도)
- **인스타 캡션 1개** (감성 + 해시태그)
- **맘카페 자연글 1개** (정보형 + 자연스러운 추천)
- **광고 카피 세트** (헤드라인 + 서브 + CTA)

## KPI 목표 설정

| 지표 | 1개월 | 2개월 | 3개월 |
|------|-------|-------|-------|
| 매출 (보수적) | 사용자 입력 | | |
| 매출 (기본) | 사용자 입력 | | |
| 매출 (낙관적) | 사용자 입력 | | |
| 리뷰 수 | 30건+ | 80건+ | 150건+ |
| 블로그 상위노출 키워드 | 5개 | 10개 | 15개 |
| 인스타 팔로워 증가 | +500 | +1,500 | +3,000 |
| 스마트스토어 클릭수 | 사용자 입력 | | |
| 맘카페 언급 수 | 10건+ | 30건+ | 50건+ |
| ROAS (광고 투자 대비) | 200%+ | 300%+ | 400%+ |

매출 목표는 PHASE 7 손익분기 분석의 시나리오를 기반으로 설정.
사용자가 구체적 수치를 모르면 시장 데이터 기반으로 추정치 제안.

## 예산 배분 가이드

마케팅 총 예산을 입력받아 채널별 배분 제안:

| 채널 | 비중 | 설명 |
|------|------|------|
| 배포형 블로그 | 25~30% | 검색 상위 노출 핵심 투자 |
| 체험단 | 20~25% | 리뷰 볼륨 확보 (제품 원가 포함) |
| 인스타 광고 | 15~20% | 리타겟팅 + 인지도 |
| 프로모션 할인 | 15~20% | 런칭 특가 / 세트 할인 마진 차이 |
| 기타 (촬영/디자인) | 10~15% | 상세페이지, 콘텐츠 제작비 |

예산 규모에 따라 비중 자동 조정:
- 소규모(100만원 미만): 체험단 + 맘카페 집중 (오가닉 위주)
- 중규모(100~500만원): 배포형 + 체험단 + 인스타 광고
- 대규모(500만원+): 전 채널 균등 + 네이버 쇼핑 광고 추가

## 리포트 반영

HTML 리포트의 "런칭 전략" + "액션플랜" 탭:
- **런칭 전략**: 3단계 타임라인 + 채널별 역할 카드 + KPI 대시보드
- **액션플랜**: 주차별 아코디언 (열면 채널별 구체 액션 + 콘텐츠 소재)
- **예산 배분**: 파이 차트 + 채널별 예상 ROI
- **콘텐츠 소재함**: 소구점별 카피/제목 목록 (복사 가능)
