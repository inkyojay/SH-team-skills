# PHASE 2.6 — 고객 목소리(VOC) 통합 분석 가이드

## 개요

네이버 카페글/블로그/지식인/뉴스/검색어 트렌드를 통합 분석하여 **실제 고객의 목소리**를 데이터로 수집한다.
PHASE 2.5 쇼핑 데이터 분석과 함께 실행하면 시장을 "판매자 관점(쇼핑)"과 "소비자 관점(VOC)" 양쪽에서 입체적으로 파악할 수 있다.

네이버 개발자센터 Client ID/Secret 동일하게 사용 (PHASE 2.5와 같은 키).

## 5가지 분석

### 1. 카페글(맘카페) 분석 — ⭐ 가장 핵심

API: `/v1/search/cafearticle.json`

맘카페에서 실제 부모들이 올린 글을 100건 수집하여:
- **주요 활동 카페** TOP 10 (어떤 맘카페에서 가장 활발한지)
- **키워드 빈도** TOP 30 (실제 고객 언어 추출)
- **목소리 패턴 분류**:
  - 고민/질문: "추천해주세요", "뭐가 좋아요" → 구매 전 고객 니즈
  - 불만/단점: "보풀", "미끄러움", "안 시원" → 제품 개선 기회
  - 만족/장점: "시원해요", "꿀잠" → 마케팅 카피 소재
  - 구매/가격: "공구", "할인", "가성비" → 프로모션 전략 근거

검색 쿼리 패턴:
- "[카테고리] 추천" → 구매 의도 글 수집
- "[카테고리] 후기" → 사용 경험 글 수집
- "[카테고리] 고민" → 고객 페인포인트 수집

### 2. 블로그 리뷰 분석

API: `/v1/search/blog.json`

체험단/리뷰 블로그를 분석하여:
- **리뷰 유형 분류**: 체험단/협찬 vs 진성 리뷰 추정 비율
- **리뷰 키워드** TOP 30 (가장 많이 언급되는 특성)
- **활발한 블로거** TOP 10 (인플루언서 후보 파악)
- 경쟁사별 리뷰 수 비교 → 마케팅 활동량 간접 측정

검색 쿼리: "[카테고리] 후기 리뷰"

### 3. 지식인 질문 분석

API: `/v1/search/kin.json`

구매 전 고객의 질문 패턴을 분석하여:
- **질문 유형 분류**: 추천 요청 / 비교 질문 / 안전성 우려 / 사용 시기 / 소재 질문 / 가격 질문 / 사용 방법
- **롱테일 키워드 후보** 자동 추출: 질문 제목 → PHASE 4 키워드 발굴에 직접 입력
- 가장 많이 궁금해하는 것 = **상세페이지에 반드시 답해야 할 내용**

### 4. 뉴스 동향 수집

API: `/v1/search/news.json`

최신 뉴스에서:
- 업계 동향, 신소재 뉴스
- 경쟁사 보도자료, 신제품 출시 뉴스
- 규제/안전 관련 이슈
- 시장 성장/축소 기사

### 5. 데이터랩 검색어 트렌드

API: `/v1/datalab/search`

네이버 통합검색에서 키워드의 **상대적 검색량 추이** (0~100 스케일):
- 최대 5개 키워드 동시 비교
- 월별 추이로 성수기/비수기 파악
- 전년 동기 대비 성장/하락 판단
- 쇼핑인사이트(PHASE 2.5)와 교차 검증

## 실행 스크립트

`scripts/naver_voc_analysis.py` — 6가지 모드:

```bash
# 전체 통합 분석 (한 번에 5가지 전부)
python naver_voc_analysis.py full \
  --client-id "YOUR_ID" --client-secret "YOUR_SECRET" \
  --query "아기 쿨매트" \
  --trend-keywords "아기 쿨매트,아기 냉감패드,아기 여름이불" \
  --output voc_results.json

# 개별 실행
python naver_voc_analysis.py cafe --query "아기 쿨매트 추천" ...
python naver_voc_analysis.py blog --query "아기 쿨매트 후기" ...
python naver_voc_analysis.py kin  --query "아기 쿨매트" ...
python naver_voc_analysis.py news --query "아기 냉감 수면용품" ...
python naver_voc_analysis.py trend --keywords "아기 쿨매트,냉감패드" ...
```

## Claude 환경에서의 대안

API 차단 시: web_search + web_fetch로 네이버 카페/블로그/지식인 검색 결과를 수집하여 유사 분석 수행. 정확도는 떨어지지만 핵심 인사이트 도출은 가능.

## 리포트 통합

VOC 분석 결과는 HTML 리포트의 다음 탭에 반영:
- **시장 분석 탭**: 카페 목소리 패턴 + 검색어 트렌드 차트
- **경쟁사 분석 탭**: 블로그 리뷰 경쟁사 비교 + 브랜드 언급 빈도
- **키워드 분석 탭**: 지식인 질문에서 추출한 롱테일 키워드 후보
- **사업 타당성 탭**: "고객의 X%가 이런 불만을 가지고 있다" 데이터 근거
- **레퍼런스 보드**: 뉴스 동향 요약
