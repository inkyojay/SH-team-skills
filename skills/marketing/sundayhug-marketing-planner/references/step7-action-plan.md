# STEP 7 — 기간별 액션플랜 + KPI 생성 가이드

## 액션플랜 생성 로직

기간(시작월~종료월)을 입력받아 아래 구조로 자동 생성한다.

---

## 월별 테마 배정 원칙

### 3개월 기준 템플릿

| 월차 | 테마 | 핵심 활동 | 배포형 목표 |
|-----|------|---------|-----------|
| 1개월차 | 기반 구축 | 채널 셋업 + 체험단 1기 + SEO 기반 발행 | 4건 |
| 2개월차 | 시즌/이슈 선점 | 시즌 키워드 집중 + 체험단 2기 + 캠페인 론칭 | 6건 |
| 3개월차 | 바이럴 수확 | 공동구매 + UGC 자생 + SOP 스킬화 | 4건 |

### 시즌별 조정

**봄 시즌 (3~5월)**
- 1월차: 기반 구축 + 뒤집기/속싸개 졸업 키워드
- 2월차: 어린이날 선물 + 초여름 태열 선행 SEO
- 3월차: 여름 시즌 선점 완료 + 공동구매 준비

**여름 시즌 (6~8월)**
- 1월차: 에어컨+태열 키워드 피크 공략
- 2월차: 메쉬 슬리핑백 전환 수요 최대화
- 3월차: 가을 밤부 선행 SEO + 여름 UGC 수확

**가을 시즌 (9~11월)**
- 1월차: 간절기 → 겨울 전환 키워드
- 2월차: 트리플밤부 피크 + 돌 선물 시즌
- 3월차: 겨울 선행 SEO + SOP 정리

**겨울 시즌 (12~2월)**
- 1월차: 트리플밤부 피크 + 보온 키워드
- 2월차: 설 선물 + 신생아 시즌
- 3월차: 봄 선행 SEO + 연간 성과 정리

## 주차별 액션 생성 규칙

각 주차는 5개 채널(N/I/T/C/P)별로 1~2개 액션으로 구성:

```
채널 코드:
N = 네이버 블로그 / 스마트스토어
I = 인스타그램
T = 스레드
C = 맘카페 (체험단 포함)
P = 배포형 블로그 (파워블로거)
```

**주차별 필수 액션 템플릿**:

| 주차 | N | I | T | C | P |
|-----|---|---|---|---|---|
| 1주 | 채널 최적화 or 블로그 발행 | 피드/릴스 | 첫 글 or 정보글 | 체험단 모집/발송 | 섭외+원고 전달 |
| 2주 | 블로그 발행 + 체험단 후기 | 릴스 + 스토리 | 공감 질문글 | 후기 발행 시작 | 발행 확인 + 순위 체크 |
| 3주 | 블로그 발행 + UGC 관리 | 캠페인 or UGC 수집 | 정보 바이럴 글 | 댓글 참여 확대 | 추가 발행 |
| 4주 | 성과 점검 + 다음달 선행 | A/B 분석 | 공동구매 타진 | 다음 단계 준비 | 다음달 원고 준비 |

---

## 집중 키워드 배정 방식

월 시작 시 해당 월의 "집중 키워드 3개"를 먼저 확정:
1. 시즌 키워드 1개 (그 달 검색량 가장 높을 것)
2. 전환 키워드 1개 (구매 의도 높은 것)
3. 롱테일 키워드 1개 (경쟁 낮고 SEO 장기 자산)

이 3개를 그 달 모든 채널 콘텐츠에 반복 노출한다.

## 월말 성과 점검 체크리스트

```
□ 네이버: 집중 키워드 3개 순위 확인 (목표: 20위 이내)
□ 네이버: 브랜드 블로그 월간 방문자 수 확인
□ 스마트스토어: 블로그/배포형 유입 경로 비율 확인
□ 인스타: 릴스 저장율 + 팔로워 증가 확인
□ 맘카페: 후기 글 저장수 + 댓글 반응 확인
□ 배포형: 건당 키워드 순위 변동 분석 → ROI 판단
□ 다음달: 효과 좋은 채널 비중 확대 / 비효율 채널 교체
```

---

## 3개월 후 SOP 스킬화 체크리스트

```
□ 잘 된 콘텐츠 포맷 목록화 (릴스 구성, 블로그 제목 패턴)
□ 체험단 모집 → 선발 → 발송 → 가이드 → 발행 → 관리 SOP 문서화
□ 파워블로거 우수 채널 목록 + 건당 효과 기록
□ 키워드별 배포형 ROI 기록표 작성
□ 다음 시즌/제품 즉시 적용 가능한 콘텐츠 템플릿 정리
□ 해시태그 세트 (채널별) 정리
```
