# STEP 6 — 비교 콘텐츠 전략 가이드

## 비교 콘텐츠가 강력한 이유

1. 구매 결정 직전 단계 타겟 → 전환율 높음
2. "A vs B" 형식은 공유·저장 확산에 유리
3. 경쟁 키워드보다 롱테일 비교 키워드가 경쟁 낮음
4. 썬데이허그가 프레임을 주도하면 결론 통제 가능

---

## 6가지 비교 유형

### 유형 1 — 제품 카테고리 비교
**주제**: 슬리핑백 vs 이불
**프레임**: "아이 수면 안전, 어떤 게 나을까요?"
**우리 포지션**: 낙상 방지, 체온 유지, 밤새 보호
**비교 대상**: 일반 이불, 좁쌀이불, 모슬린 이불
**결론 유도**: "이불은 걷어찰 수 있지만 슬리핑백은 못 벗어나요"
**채널**: 배포형 블로그 + 맘카페 정보글

### 유형 2 — 발달 단계 전환 비교
**주제**: 속싸개 vs 슬리핑백
**프레임**: "뒤집기 전후, 무엇을 입혀야 할까요?"
**우리 포지션**: 뒤집기 시작 = 슬리핑백으로 전환 시점
**비교 대상**: 속싸개 전반
**결론 유도**: "뒤집기 후 속싸개는 위험할 수 있어요"
**채널**: 배포형 1순위 + 블로그 + 인스타 카드뉴스

### 유형 3 — 소재 간 비교 (우리 라인업 내)
**주제**: 메쉬 vs 밤부 vs 트리플밤부
**프레임**: "어떤 소재가 우리 아기에게 맞을까요?"
**우리 포지션**: 계절별 라인업 완비 (모든 답이 썬데이허그)
**비교 대상**: 타사 단일 소재 제품
**결론 유도**: "계절마다 맞는 소재가 달라요. 썬데이허그엔 세 가지 다 있어요"
**채널**: 블로그 심층글 + 스마트스토어 상세페이지

### 유형 4 — 원산지 비교
**주제**: 국내산 vs 수입산 아기 수면조끼
**프레임**: "아기 용품, 원산지 정말 중요할까요?"
**우리 포지션**: 100% 국내 생산, 소재 투명성, 품질 관리
**비교 대상**: 수입 브랜드 슬리핑백
**결론 유도**: "국내산은 소재 확인과 A/S가 다릅니다"
**채널**: 배포형 블로그 + 맘카페 신뢰 글

### 유형 5 — 브랜드 간 비교
**주제**: 슬리핑백 브랜드 A vs B vs 썬데이허그
**프레임**: "실제 써본 엄마가 비교한 슬리핑백 후기"
**우리 포지션**: 소재 다양성 + 사이즈 범위 + Made in Korea
**비교 대상**: 알쏠비, 러브레터 등 경쟁 브랜드
**결론 유도**: "국내산 + 메쉬·밤부·트리플 3종 라인업 = 썬데이허그"
**채널**: 파워블로거 의뢰 (브랜드 계정 직접 발행 금지)

### 유형 6 — 가격대 비교
**주제**: 3만원대 수면조끼 vs 5만원대 수면조끼
**프레임**: "비싼 슬리핑백, 정말 다를까요?"
**우리 포지션**: 국내산 프리미엄 포지셔닝, 소재·A/S 차별화
**비교 대상**: 저가 수입 제품
**결론 유도**: "아기 피부에 닿는 것, 조금 더 투자할 이유가 있어요"
**채널**: 배포형 블로그 + 스레드 질문 답변

## 비교 콘텐츠 작성 원칙

### 공정성 유지 (광고처럼 안 보이기)
```
✓ 경쟁 제품의 장점도 인정
  "다른 브랜드도 소재는 좋아요. 다만..."

✓ 상황별 추천으로 중립 포지션
  "태열이 심하지 않다면 다른 제품도 괜찮을 수 있어요.
   다만 한여름 에어컨 환경이라면..."

✓ 우리 제품이 강한 항목을 마지막에
  비교표에서 마지막 항목이 = 국내산 / 소재 다양성 / 투웨이 지퍼

✗ "우리 제품이 최고입니다" 단정 금지
✗ 경쟁 제품을 깎아내리는 표현 금지
```

### 콘텐츠 필수 구성 요소
```
□ 비교 표 (소재 / 가격 / 원산지 / 사이즈 / 특징)
□ 실제 사용 사진 (자연광, 착용 상태)
□ 구체적 수치 (소재 혼방 비율, 사이즈 cm, 가격)
□ 상황별 추천 정리 (여름/겨울, 태열/일반 등)
□ 자연스러운 결론 (썬데이허그 링크 포함)
```

### 채널별 비교 콘텐츠 형태
| 채널 | 형태 | 분량 |
|------|------|------|
| 네이버 블로그 | 비교 심층 글 | 2,000자+ + 비교표 |
| 배포형 블로그 | 동일 원고 파워블로거에 배포 | 1,500자+ |
| 인스타그램 | 카드뉴스 (5~8장) | 장당 50자 이내 |
| 맘카페 | 정보형 후기 글 | 1,000자+ + 사진 3장 |
| 스레드 | 대화형 질문 답변 | 200자 이내 |
