# 배포형 블로그 전략 세부 가이드

## 왜 배포형이 필요한가

브랜드 블로그는 신규 도메인이라 네이버 신뢰도가 낮다.
경쟁 키워드에서 1페이지 진입이 어렵고 시간이 많이 걸린다.

파워블로거·체험단 채널은 이미 네이버 최적화 블로그라 동일한 콘텐츠도 더 빠르게, 더 높은 위치에 노출된다.

---

## 배포형 4가지 타입

### Type 1 — 파워블로거 원고 의뢰 (유료)
- **특징**: 네이버 최적화 블로그에 원고 의뢰
- **비용**: 건당 8~15만원
- **플랫폼**: 픽업엔, 애드픽, 블로그 대행사 직접 컨택
- **적합 키워드**: 경쟁 중간~높음, 빠른 노출 필요
- **원고 작성**: 브랜드에서 직접 작성 후 전달 (품질 통제)

### Type 2 — 자체 체험단 블로그 (저비용)
- **특징**: 제품 원가만으로 운영
- **비용**: 제품 원가 (2~5만원 수준)
- **적합 키워드**: 경쟁 낮음, 진성 후기 필요
- **장점**: 신뢰도 높은 진성 콘텐츠
- **단점**: 퀄리티 편차, 키워드 노출 불확실

### Type 3 — 비교 특화 배포
- **특징**: 비교 탐색 키워드 전용 콘텐츠
- **원고 형태**: 브랜드에서 직접 작성한 비교 원고를 파워블로거에 의뢰
- **핵심**: 공정해 보이되 결론은 썬데이허그로
- **적합 키워드**: "슬리핑백 브랜드 비교", "속싸개 vs 슬리핑백" 등

### Type 4 — 육아 미디어 기고 (브랜드 공신력)
- **특징**: 베이비뉴스, 맘앤앙팡, 월간유아 등
- **비용**: 무료 기고 ~ 유료 보도자료
- **효과**: 네이버 뉴스 탭 노출 + 브랜드 공신력
- **활용**: 수면 안전 가이드, 육아 전문가 칼럼 형태

## 파워블로거 의뢰 SOP

### 1. 원고 사전 작성
브랜드에서 먼저 완성된 원고 작성. 파워블로거에게 원고 수정 최소화 지시.

원고 필수 포함 요소:
```
□ 타겟 키워드 1~2개 (제목 + 본문 2회 이상)
□ 구체적 수치 (온도, 사이즈, 가격 등)
□ 비교 표 또는 가이드 표
□ 스마트스토어 링크 (자연스럽게 1~2곳)
□ 실제 사용 사진 (제공 or 파워블로거 자체 촬영)
□ 분량: 1,500자 이상
```

### 2. 파워블로거 섭외
- 픽업엔: https://www.picupn.com
- 직접 컨택: 인스타 "육아 블로그" 검색 후 이웃 수·게시물 수 확인
- 조건 확인: 블로그 이웃 수 1,000+ / 최근 발행 이력 활발 / 네이버 최적화 확인

### 3. 발행 후 모니터링
- 발행 즉시: 네이버 해당 키워드 검색 → 위치 확인
- 2주 후: 키워드 순위 변동 확인
- 목표: 10~20위 이내 진입

## 3개월 배포형 예산 가이드

| 기간 | 주제 | 건수 | 예산 |
|------|------|------|------|
| 1개월차 | 기반 키워드 (뒤집기, 슬리핑백 비교) | 4건 | 40만원 |
| 2개월차 | 시즌 키워드 (에어컨, 태열, 선물) | 6건 | 55만원 |
| 3개월차 | 전환 키워드 (브랜드 비교, 수면루틴) | 4건 | 40만원 |
| **합계** | | **14건** | **135만원** |

**ROI 측정 방법**:
1. 발행 전: 해당 키워드 현재 순위 기록
2. 발행 2주 후: 순위 재확인
3. 발행 1개월 후: 스마트스토어 유입 경로에서 해당 블로그 트래픽 확인
4. 효과 없는 채널 → 교체, 효과 좋은 채널 → 반복 의뢰

## 배포형 원고 우선순위 리스트

새로운 제품 적용 시 아래 우선순위로 원고 배포:

**1순위 (즉시 발행)**
- "[제품 전환 시점] 완벽 가이드" — 뒤집기, 첫 구매 등
- "[제품] vs [대체품]" 비교 — 이불, 속싸개 등

**2순위 (시즌 2~4주 전 발행)**
- "[시즌] + [증상] + 해결 가이드" — 여름 태열, 겨울 보온 등
- "출산 선물 / [이벤트] 선물 추천" — 어린이날, 출산 등

**3순위 (롱테일 SEO 자산)**
- "[소재] 특징과 선택 가이드"
- "국내산 아기 [제품] 브랜드 비교"
- "수면교육 + [제품] 효과"
