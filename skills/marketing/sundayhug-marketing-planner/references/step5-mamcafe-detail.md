# 맘카페 바이럴 전략 세부 가이드

## 핵심 원칙

**브랜드 계정으로 직접 참여 절대 금지.**
맘카페는 광고가 들키는 순간 역효과 + 계정 정지 위험.
체험단이 실제 엄마·아빠로 자연스럽게 녹아드는 것이 전부.

---

## 4가지 바이럴 전략

### 전략 1 — 체험단 진성 후기 발행

**체험단 선발 기준**:
- 맘카페 활동 이력 있는 분 우선 (게시글 5개 이상)
- 카페 등급 정회원 이상
- 인스타·블로그 병행 가능한 분 우대
- 실제 해당 연령대 아이 있는 분 (제품 카테고리 맞는 아이)

**가이드 키트 필수 구성**:
```
□ 후기 주제 3가지 중 선택
  - "뒤집기 시작하고 슬리핑백으로 바꿨어요"
  - "태열 아기 여름 수면 고민 해결한 제품"
  - "수면교육 3개월째 — 슬리핑백 효과 진짜?"
□ 필수 키워드 리스트 (자연스럽게 1~2개 포함)
□ 사진 가이드: 자연광 / 아기 착용 / 소재 클로즈업
□ 맘카페 후기 팁: 스토리 형태로, 광고 말미에 언급
□ 분량 가이드: 800자 이상, 사진 3장 이상
```

**발행 후 관리**:
- 발행 즉시 공감·댓글로 초기 반응 활성화 지원
- 스마트스토어 링크 삽입 확인
- 키워드 포함 여부 확인

---

### 전략 2 — 질문글 댓글 자연 참여

**진입 키워드 (검색해서 찾기)**:
- "수면조끼 추천해주세요"
- "슬리핑백 뭐 써요"
- "뒤집기 시작했는데 이불 어떻게 해요"
- "이불 차는 아기 어떻게 해요"
- "태열 아기 잠옷 추천"

**댓글 진입 방식** (절대 광고처럼 쓰지 말 것):
```
❌ 나쁜 예: "썬데이허그 슬리핑백 써보세요! 좋아요!"
✓ 좋은 예: "저도 비슷한 고민으로 한참 찾다가 슬리핑백으로 바꿨어요.
           우리 아이가 뒤집기 시작하고 나서 썼는데 확실히 달라지더라고요.
           혹시 코튼 메쉬 슬리핑백 알아보셨어요? 태열 있는 아기한테 괜찮았어요"
```

**주 3~5개 글 목표**. 체험단 1인당 주 1~2개 참여.

---

### 전략 3 — 정보글 바이럴 (저장수 특화)

맘카페에서 가장 강력한 바이럴 형태. 저장수 50+ 글은 입소문 자생.

**저장 유도 글 공식**:
```
제목: [유용한 정보] + "(저장필수)" 또는 "(정리해봤어요)"
예: "실내온도별 아기 착용 가이드 총정리 (저장필수)"

본문 구성:
1. 공감 도입 (2~3줄)
2. 정보 테이블 (온도별/상황별)
3. 실제 적용 팁 (3~5개)
4. 마무리: "저는 썬데이허그 코튼 메쉬로 해결했어요" (1줄 자연 언급)

필수 포함: 표 / 이미지 / 구체적 수치
분량: 1,000자 이상
```

---

### 전략 4 — 공동구매 제휴

**공구 타이밍**: 시즌 피크 1~2개월 전 준비, 피크 시작 직후 오픈

**공구 조건 설계**:
- 카페 전용 할인코드: 10~15%
- 번들 특가: 3장 세트 묶음
- D-day 카운트다운 긴박감

**담당자 컨택 방법**:
1. 카페 공지사항에서 "공동구매 신청" 게시물 찾기
2. 운영자 쪽지 또는 이메일 컨택
3. 제안서: 브랜드 소개 + 제품 USP + 할인 조건 + 배송 일정

## 주요 타겟 카페 채널

| 우선순위 | 카페명 | 특징 | 공략 게시판 |
|--------|-------|------|-----------|
| 최우선 | 맘스홀릭베이비 | 네이버 최대 육아 카페 | 수면·육아용품 게시판 |
| 최우선 | 미즈넷 | 30대 초반 실구매층 | 제품 후기 게시판 |
| 최우선 | 임신육아 | 임신~육아 전반 | 육아 고민 게시판 |
| 2순위 | 아기랑 | 0~12개월 집중 | 월령별 게시판 |
| 2순위 | 베이비트리 | 출산 선물 탐색층 | 육아용품 추천 |
| 2순위 | 지역 맘카페 | 공동구매 연계 | 공구 게시판 |

## 체험단 운영 주의사항

```
✓ 해야 할 것
- 후기 말미에 "체험단 제공받아 작성했습니다" 자연스럽게 언급 (법적 의무)
- 썬데이허그보다 아기의 스토리를 주인공으로
- 질문형 댓글 참여 시 "저도 비슷한 고민으로..." 자연 진입
- 사진은 자연광 + 실제 아기 착용 필수

✗ 하지 말 것
- 브랜드 계정으로 직접 홍보글 작성 (계정 정지 위험)
- 여러 계정으로 같은 글에 동시 댓글 (어뷰징 신고)
- "~하세요", "추천드려요" 등 권유형 직접 표현
- 체험단임을 글 서두에 언급 (글 첫 번째 문장 금지)
```
