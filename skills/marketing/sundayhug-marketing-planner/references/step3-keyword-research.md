# STEP 3 — 롱테일 키워드 발굴 가이드

## 키워드 발굴 프레임워크

### 씨앗 키워드에서 롱테일 확장하기

**씨앗 키워드** (제품 카테고리 핵심어):
- 슬리핑백 / 수면조끼 / 입는이불
- 속싸개 / 아기이불

**확장 방향 6가지**:
1. **증상/고통** + 씨앗 = "아기 이불 걷어차는 + [씨앗]"
2. **시즌/온도** + 씨앗 = "여름 / 에어컨 / 태열 + [씨앗]"
3. **발달 단계** + 씨앗 = "뒤집기 후 / 100일 / 신생아 + [씨앗]"
4. **소재** + 씨앗 = "코튼 메쉬 / 밤부 / 국내산 + [씨앗]"
5. **목적** + 씨앗 = "수면교육 / 수면독립 / 선물 + [씨앗]"
6. **비교** = "[씨앗] vs [대체품]" / "[씨앗] 브랜드 비교"

## 경쟁도 판단 기준

| 경쟁도 | 판단 기준 | 전략 |
|--------|---------|------|
| 낮음 (공략) | 3단어 이상, 구체적 상황 포함 | 브랜드 블로그로도 상위 가능 |
| 중간 | 2단어, 일반적 표현 | 배포형 블로그로 보강 |
| 높음 | 1단어, 범용 표현 | 배포형 집중 or 포기 |

## 제품별 시즌 키워드 패턴

### 여름 시즌 (5~8월) — 메쉬 슬리핑백 집중
- 에어컨 켜고 아기 재울 때 옷
- 여름 아기 에어컨 온도 몇 도
- 태열 아기 여름 수면 온도
- 여름 신생아 잠옷 추천
- 에어컨 바람 아기 배 차가운 거 해결

### 겨울 시즌 (11~2월) — 트리플밤부 집중
- 겨울 아기 슬리핑백 보온
- 아기 추울 때 수면복 추천
- 신생아 겨울 수면환경 온도
- 보온 수면조끼 추천

### 봄/가을 간절기 (3~4월, 9~10월) — 밤부코튼 집중
- 간절기 아기 수면 온도
- 봄 아기 잠옷 레이어링
- 일교차 큰 날 아기 수면복

### 연중 공통
- 뒤집기 후 속싸개 졸업 시기
- 수면교육 슬리핑백 효과
- 아기 슬리핑백 사이즈 고르는 법
- 슬리핑백 언제부터 사용하나요
- 아기 수면 루틴 만들기
- 출산 선물 아기 수면용품

## 채널별 키워드 형태 차이

### 네이버 블로그/스마트스토어
- 자연어 질문형: "뒤집기 후 속싸개 졸업 시기"
- 명사 나열형: "태열 아기 여름 수면조끼 추천"
- 긴 꼬리형: "에어컨 켜고 아기 재울 때 입히는 옷"

### 인스타그램 해시태그
- 압축형: #아기슬리핑백 #수면조끼추천
- 상황형: #뒤집기후필수템 #태열아기필수템
- 감성형: #꿀잠루틴 #아기수면교육 #수면독립

### 스레드
- 질문/대화형: "뒤집기 이후 수면 어떻게 해요?"
- 공감형: "아기 이불 걷어차는 거 해결법"
- 후기 요청형: "슬리핑백 어느 브랜드 쓰세요?"

### 맘카페
- 질문글 제목형: "수면조끼 추천해주세요"
- 후기형: "슬리핑백으로 바꿨어요 — 뒤집기 후기"
- 정보형: "실내온도별 아기 착용 가이드 (저장필수)"

```
키워드: [키워드 텍스트]
의도: 문제해결 / 구매의도 / 비교탐색 / 정보탐색
경쟁도: 높음 / 중간 / 낮음
활용: [구체적 콘텐츠 형태와 채널]
```
