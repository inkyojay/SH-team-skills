# STEP 4 — 콘텐츠 우선순위 전략 가이드

## 우선순위 배분 원칙

키워드를 3개 버킷으로 나눠 채널·예산·타이밍을 다르게 운영한다.

---

## 1순위 버킷 — 문제해결형 (전환율 최고)

**특징**: 명확한 고통 포인트. 구매 의도가 이미 있음. 경쟁 낮음.

**전략**: 모든 채널에 동시 다발 투하. 같은 메시지를 네이버·맘카페·스레드·배포형에서 동시에 노출해 검색자가 어디서 만나든 썬데이허그로 귀결되게.

**콘텐츠 공식**:
```
제목: [고통 상황] + [해결 방법] + [가이드/방법]
예: "뒤집기 시작했나요? 속싸개 졸업 시기와 슬리핑백 전환 완벽 가이드"
구조: 고통 공감 → 원인 설명 → 해결책 → 제품 자연 연결
길이: 블로그 2,000자+ / 릴스 60초 이내 / 맘카페 1,000자+
```

**필수 포함 요소**:
- 온도별/상황별 착용 가이드 표 (저장 유도)
- 실제 사용 아기 사진
- 구체적 수치 (실내 온도 몇 도, 개월 수)

---

## 2순위 버킷 — 정보탐색형 (SEO 장기 자산)

**특징**: 구매 의도 없이 정보만 찾는 단계. 시리즈로 묶으면 내부 링크 SEO 시너지.

**전략**: 블로그 시리즈 3~5편. 각 편이 서로 내부 링크로 연결. 배포형에서도 같은 시리즈 추가 발행.

**콘텐츠 공식**:
```
시리즈 구성 예시 — "아기 수면 완벽 가이드" 시리즈:
  1편: 수면교육 언제부터? 기초 가이드
  2편: 슬리핑백이 수면 루틴에 도움이 될까?
  3편: 아기 수면 온도와 착용 가이드
  4편: 슬리핑백 사이즈 선택 완벽 정리
  5편: 슬리핑백 관리법 — 세탁부터 보관까지
```

각 편 말미에 다음 편 링크 + 스마트스토어 링크 배치.

---

## 3순위 버킷 — 구매의도형 (직접 전환)

**특징**: "추천해줘"라고 검색하는 사람. 구매 준비 완료 상태.

**전략**: 배포형 블로그(파워블로거) 집중 공략. 스마트스토어 키워드 태그 강화.

**콘텐츠 공식**:
```
제목: [구체적 조건] + 추천 / BEST / 후기
예: "코튼 메쉬 아기 슬리핑백 추천 — 태열 있는 아기 엄마가 직접 골라봤어요"
구조: 선발 기준 → 비교 → 추천 TOP 3 → 썬데이허그 강조
```

---

## 비교 콘텐츠 — 별도 전략

비교탐색형 키워드는 별도 섹션(STEP 6)으로 관리.
구매 직전 단계 타겟이라 전환율 매우 높음.
반드시 파워블로거 의뢰 대상에 포함시킬 것.

---

## 콘텐츠 캘린더 배분 원칙

월간 발행 목표 기준 배분:

| 버킷 | 브랜드 블로그 | 체험단/UGC | 배포형 |
|------|------------|----------|--------|
| 1순위 (문제해결) | 40% | 50% | 40% |
| 2순위 (정보탐색) | 40% | 30% | 30% |
| 3순위 (구매의도) | 20% | 20% | 30% |

시즌 피크(여름/겨울)에는 1순위 시즌 키워드 비중을 60~70%로 높인다.
