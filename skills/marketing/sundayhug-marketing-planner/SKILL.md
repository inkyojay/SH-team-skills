# 썬데이허그 마케팅 플래너 스킬

이 스킬은 썬데이허그(JAYCORP) 제품에 대한 마케팅 전략 전체를 표준화된 프로세스로 반복 생성한다.
신제품이 나올 때마다, 또는 기존 제품의 시즌 전략이 필요할 때마다 즉시 적용 가능하다.

## 브랜드 컨텍스트

- **브랜드명**: 썬데이허그 (Sunday Hug)
- **회사**: 주식회사 제이코프 (JAYCORP)
- **카테고리**: 아기 수면 용품 (슬리핑백, 속싸개 등)
- **핵심 포지셔닝**: "입는 이불" — 아기 꿀잠을 위한 국내산 프리미엄 수면 용품
- **주요 타겟**: 0~5세 자녀를 둔 30대 부모 (특히 엄마)
- **채널 구성**: 네이버 블로그/스마트스토어 + 인스타그램 + 스레드 + 맘카페 + 배포형 블로그

---

## 실행 프로세스 — 7단계

아래 7단계를 순서대로 실행한다. 각 단계의 세부 가이드는 `references/` 파일에 있다.

### STEP 1 — 제품 정보 수집
→ 세부 가이드: `references/step1-product-research.md`

- 제품 URL이 있으면 `web_fetch`로 상세페이지 크롤링
- 없으면 사용자에게 제품명, 소재, 가격, 사이즈, 주요 특징 직접 입력 요청
- 리뷰 텍스트(실제 구매자 언어)도 수집 — 키워드 발굴의 금맥

**수집 항목**: 제품명 / 가격 / 소재 / 사이즈 / 주요 기능 / 시즌 / 리뷰 키워드

---

### STEP 2 — 제품 USP 분석
→ 세부 가이드: `references/step2-usp-analysis.md`

6가지 차원으로 USP를 도출한다:
1. **기능적 차별점** (소재, 구조, 특허)
2. **안전성/신뢰** (국내산, 인증, 소재 투명성)
3. **편의성** (투웨이 지퍼, 탈의 방지 등)
4. **성장 대응** (사이즈 범위, 단계별 설계)
5. **감성적 가치** (수면 루틴, 부모 안심)
6. **시즌/온도 대응** (여름 메쉬, 겨울 트리플 등)

출력: USP 카드 6개 + 헤드라인 카피 5개 + 타겟 구매 시나리오 4개

---

### STEP 3 — 롱테일 키워드 발굴
→ 세부 가이드: `references/step3-keyword-research.md`

4가지 검색 의도로 분류해 30개 이상 발굴:

| 의도 유형 | 설명 | 전환율 |
|---------|------|--------|
| 문제해결형 | "아기 이불 걷어차는 이유" | 최고 |
| 구매의도형 | "코튼 메쉬 슬리핑백 추천" | 높음 |
| 비교탐색형 | "슬리핑백 vs 이불" | 높음 |
| 정보탐색형 | "수면교육 슬리핑백 효과" | 중간 |

각 키워드에 경쟁도(높음/중간/낮음)와 활용 채널 매핑 포함.
**낮음 경쟁도 + 구매/문제해결 의도 = 1순위 공략**

---

### STEP 4 — 콘텐츠 우선순위 전략
→ 세부 가이드: `references/step4-content-strategy.md`

키워드를 3개 우선순위 그룹으로 묶어 채널 배분:

- **1순위 (전환율 최고)**: 문제해결형 키워드 묶음 → 블로그+배포형+맘카페 동시 투하
- **2순위 (SEO 자산)**: 정보탐색형 시리즈 → 블로그 3~5편 연속 + 내부 링크
- **3순위 (직접 전환)**: 구매의도형 → 배포형 블로그 + 스마트스토어 태그

각 묶음마다: 타겟 키워드 / 콘텐츠 제목 예시 / 발행 채널 / 주의사항

---

### STEP 5 — 채널별 콘텐츠 플랜
→ 세부 가이드: `references/step5-channel-plan.md`

5개 채널 각각의 역할과 실행 방법:

**네이버 블로그/스마트스토어** — SEO 장기 자산
- 브랜드 발행: 월 4회 정보형 롱폼 (1,500자+)
- 체험단 UGC: 월 8~12건 진성 후기
- 스마트스토어: 키워드 최적화 + 포토리뷰 유도

**인스타그램** — 감성 브랜딩 + 바이럴
- 피드: 주 3회 라이프스타일 컷
- 릴스: 주 2회 정보형 숏폼 (저장 유도)
- UGC 캠페인: #썬데이허그꿀잠 해시태그

**스레드** — 신뢰 기반 커뮤니티 진입
- 육아 꿀팁 정보글 주 3~4회
- 투표/질문형 소통으로 팔로워 확보
- 광고 느낌 없이 사람 계정처럼 운영

**맘카페** — 진성 바이럴 (핵심 채널)
→ 세부 전략: `references/step5-mamcafe-detail.md`
- 체험단 진성 후기 + 질문글 댓글 참여
- 정보글 바이럴 (저장수 50+ 목표)
- 공동구매 제휴 (시즌별 1회)

**배포형 블로그** — 경쟁 키워드 상위 노출
→ 세부 전략: `references/step5-distribution-detail.md`
- 파워블로거 원고 의뢰 (건당 8~15만원)
- 자체 체험단 블로그 (원가만)
- 비교 특화 원고 집중 배포

---

### STEP 6 — 비교 콘텐츠 전략
→ 세부 가이드: `references/step6-comparison-content.md`

"A vs B" 형식의 비교 콘텐츠 6가지 유형:
1. 슬리핑백 vs 이불
2. 속싸개 vs 슬리핑백 (뒤집기 시점)
3. 소재별 비교 (메쉬 vs 밤부 vs 트리플)
4. 국내산 vs 수입산
5. 브랜드 비교 (공정하게 시작해 우리로 귀결)
6. 가격대별 비교 (프리미엄 포지셔닝)

**핵심 원칙**: 공정해 보이되 결론은 우리로. 비교표 + 실사진 + 구체적 수치 3박자 필수.

---

### STEP 7 — 기간별 주차 액션플랜 + KPI
→ 세부 가이드: `references/step7-action-plan.md`

기간(예: 4~6월)을 입력받아 월별 테마와 주차별 액션을 생성:

| 월 | 테마 | 핵심 집중 키워드 | 배포형 목표 |
|----|------|----------------|------------|
| 1월차 | 기반 구축 | 뒤집기 후 속싸개 / 슬리핑백 언제부터 | 4건 |
| 2월차 | 시즌 선점 | 시즌 키워드 + 이슈 키워드 | 6건 |
| 3월차 | 바이럴 수확 | 전환 키워드 + 공동구매 | 4건 |

각 주차 액션은 채널별(N/I/T/C/P)로 구체적 실행 항목 명시.

---

## 출력 포맷

모든 결과물은 **에디토리얼 매거진 스타일 HTML**로 출력한다.
→ 디자인 상세 가이드: `references/html-template-guide.md`

### 디자인 핵심 요약
- **레이아웃**: 왼쪽 고정 네비게이션(220px, 잉크 배경) + 오른쪽 스크롤 기반 섹션
- **폰트**: Google Fonts — Noto Serif KR(제목) + Noto Sans KR(본문)
- **컬러**: 크림(#F7F4EF) 배경 + 잉크(#1A1714) 텍스트 + 번트오렌지(#C4571A) 강조
- **섹션 구조**: 넘버링(01, 02...) + serif 제목 + 설명문 + 카드/테이블 콘텐츠
- **인터랙션**: 스크롤 추적 네비 + 월 탭 전환 + 주차별 접기/펼치기
- **톤**: 고급스러운 에디토리얼 느낌. 보라/원색 UI 사용 금지.

### 필수 섹션 (10개)
1. 히어로 배너 (제품명 + 분기 + 칩 태그)
2. 제품 USP / 핵심 소구점 (3열 카드 그리드)
3. 타겟 구매 시나리오 (4열 박스)
4. 롱테일 키워드 30선 (의도별 테이블 + pill 뱃지)
5. 키워드별 콘텐츠 우선순위 (priority-block + 전략 서술)
6. 채널별 콘텐츠 플랜 (2열 채널 카드)
7. 맘카페 바이럴 전략 (경고 박스 + 4전략 카드)
8. 배포형 블로그 전략 (4타입 카드 + 예산 가이드)
9. 비교 콘텐츠 전략 (비교표 + 6유형)
10. 주차별 액션플랜 + KPI (월 탭 + 주차 접기 + KPI 4열 그리드)

### 파일 저장
결과물은 `output/` 디렉토리에 저장: `output/{제품명}-마케팅플랜.html`

## 빠른 실행 — 체크리스트

신제품 적용 시 이 순서로 진행:

```
□ 1. 제품 URL 또는 정보 입력
□ 2. 시즌/기간 확인 (예: 4~6월 여름 / 9~11월 가을)
□ 3. 집중할 제품 수 확인 (1개 vs 라인업 전체)
□ 4. 채널 현황 확인 (신규 채널 있는지)
□ 5. 체험단 예산/배포형 예산 확인
□ 6. STEP 1~7 순서대로 실행
□ 7. HTML 결과물 생성 + 파일 다운로드 제공
```

---

## 재사용 시 주의사항

- 시즌이 바뀌면 **STEP 3 키워드**부터 다시 실행 (여름 vs 겨울 키워드 전혀 다름)
- 제품 라인업이 다르면 **STEP 2 USP**부터 다시 실행
- 채널이 추가되면 **STEP 5**에 새 채널 추가 후 STEP 7 액션플랜 재생성
- 맘카페 전략은 **항상 브랜드 계정 직접 참여 금지** 원칙 유지
- 배포형 예산은 **월별 성과 분석 후 조정** (ROI 낮은 채널 교체)
