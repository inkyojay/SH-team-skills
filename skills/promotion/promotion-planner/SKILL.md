# 썬데이허그 프로모션 & 콘텐츠 통합 플래너

이 스킬은 썬데이허그 프로모션 전략 + 콘텐츠 발행 전략 + 마케팅 액션플랜을 통합 기획하고, 결과물을 인터랙티브 HTML 리포트로 생성한다.

## 브랜드 컨텍스트

- **브랜드**: 썬데이허그 (Sunday Hug) / 주식회사 제이코프 (JAYCORP)
- **카테고리**: 아기 수면 용품 (슬리핑백, 속싸개, 접이식 침대 등)
- **핵심 포지셔닝**: "입는 이불" — 아기 꿀잠 프리미엄 수면 용품
- **주요 타겟**: 0~5세 자녀를 둔 30대 부모 (특히 엄마)
- **운영 채널**: 자사몰(sundayhug.kr) + 네이버 스마트스토어 + 인스타그램 + 스레드 + 맘카페 + 배포형 블로그

## 실행 프로세스 — 9단계

### ★ STEP 0 — 사전 기획 대화 (반드시 먼저 실행)

→ 세부 가이드: `references/pre-planning-conversation.md`

**절대 바로 크롤링/분석부터 시작하지 않는다.**
사용자와 자연스러운 대화를 통해 기획 방향을 먼저 잡아야 한다.

#### 대화 방식

- **하나씩, 자연스럽게** 질문한다 (한꺼번에 몰아서 묻지 않음)
- 사용자의 답변에 맞장구치고, 관련 아이디어를 제안하면서 대화를 이어간다
- 각 질문에서 사용자가 "잘 모르겠어" 하면 옵션을 제시하거나 추천해준다
- 4가지 핵심 항목이 모두 확인되면 요약 후 실행 동의를 받는다

#### 확인할 4가지 핵심 항목

**① 프로모션 기간 + 예산 범위**
- 시작: "어떤 기간으로 기획할까요?" (예: 4~6월, 7~9월)
- 예산: 전체 마케팅 예산 또는 "체험단 몇 회", "배포형 몇 건" 같은 범위
- 할인 마진 허용 범위 (최대 할인율)
- 사용자가 기간만 말하면 예산은 자연스럽게 이어서 물어본다

**② 집중할 제품 / 신제품 여부**
- "이번 기간에 특별히 밀고 싶은 제품이 있나요?"
- 신제품 출시 예정이 있는지 (출시 시기 포함)
- 재고 소진이 필요한 제품이 있는지
- 없으면 웹사이트 크롤링 후 시즌에 맞는 제품을 제안해준다

**③ 쇼핑라이브 계획**
- "쇼핑라이브도 계획에 넣을까요?"
- 횟수: 월 몇 회? (없어도 OK)
- 채널: 자사몰 / 네이버 스마트스토어 / 둘 다
- 선호 요일/시간대
- 라이브 전용 혜택 범위

**④ 특별 이벤트 / 행사 일정**
- "기간 안에 특별히 연계하고 싶은 행사가 있나요?"
- 공휴일/기념일 (어린이날, 가정의 달 등)
- 자체 브랜드 이벤트 (론칭, 리뉴얼 등)
- 협업/콜라보 일정
- 사용자가 모르면 해당 기간의 시즌 이벤트를 알려주고 선택하게 한다

#### 대화 종료 조건

4가지 핵심 항목이 모두 확인되면:

1. **기획 요약을 보여준다** — "정리하면 이렇게 되네요:" 형태로 깔끔하게
2. **"이 방향으로 기획 시작할까요?"** 동의를 받는다
3. 동의 받은 후에만 STEP 1로 넘어간다

#### 대화 예시 흐름

```
Claude: "프로모션 기획 시작할게요! 먼저 기간부터 잡아볼까요? 
        어떤 기간을 생각하고 계세요?"

사용자: "4월부터 6월까지"

Claude: "좋아요, 4~6월이면 봄→여름 전환기라 소재 교체 수요가 
        크네요. 이번에 특별히 밀고 싶은 제품이 있나요? 
        아니면 신제품 출시 계획이 있으신지?"

사용자: "ABC 침대가 4월 말에 나와"

Claude: "오 ABC 침대 출시라면 큰 이벤트네요! 그러면 4월 말 
        런칭에 맞춰 기획전도 같이 잡으면 좋겠어요.
        혹시 쇼핑라이브도 계획에 넣을까요?"
        
... (자연스럽게 이어가기)
```

### STEP 2 — 월별 프로모션 테마 설정

각 월에 대해:
1. **시즌 키워드** 선정 (날씨/육아 시즌에 맞춤)
2. **메인 테마** 1개 (예: "봄 전환 소재 교체", "어린이날 선물")
3. **서브 테마** 1~2개 (보조 프로모션)
4. **집중 제품** 선정 (해당 시즌 주력 상품)

**테마 설정 원칙 (3개월 기준)**:
- 1월차: 기반 구축 + 시즌 오픈 (인지도) — 채널 셋업 + 체험단 1기 + SEO 기반
- 2월차: 이벤트 연계 피크 (전환) — 시즌 키워드 집중 + 체험단 2기 + 캠페인
- 3월차: 바이럴 수확 + 다음 시즌 예고 (재구매) — 공동구매 + UGC + SOP화

**시즌별 조정 가이드**:
→ 세부 가이드: `references/season-adjustment.md`

### STEP 4 — 콘텐츠 발행 전략 (마케팅 플래너 병합)

→ 세부 가이드: `references/content-strategy.md`

키워드 기반으로 콘텐츠를 3개 우선순위 버킷으로 분류:

| 버킷 | 키워드 유형 | 전환율 | 전략 |
|------|-----------|--------|------|
| **1순위** | 문제해결형 | 최고 | 모든 채널 동시 투하 |
| **2순위** | 정보탐색형 | 중간 | 블로그 시리즈 3~5편 + 내부링크 SEO |
| **3순위** | 구매의도형 | 높음 | 배포형 집중 + 스마트스토어 태그 |

**콘텐츠 공식**:
- 1순위: `[고통 상황] + [해결 방법] + [가이드]` → 고통 공감 → 원인 → 해결 → 제품 연결
- 2순위: 시리즈 편성 (5편) → 각 편 내부 링크 + 스마트스토어 링크
- 3순위: `[구체적 조건] + 추천 / BEST` → 선발 기준 → 비교 → TOP 3

**월간 콘텐츠 배분 비율**:

| 버킷 | 브랜드 블로그 | 체험단/UGC | 배포형 |
|------|------------|----------|--------|
| 1순위 (문제해결) | 40% | 50% | 40% |
| 2순위 (정보탐색) | 40% | 30% | 30% |
| 3순위 (구매의도) | 20% | 20% | 30% |

시즌 피크에는 1순위 시즌 키워드 비중 60~70%로 상향.

### STEP 5 — 비교 콘텐츠 전략 (마케팅 플래너 병합)

→ 세부 가이드: `references/comparison-content.md`

구매 직전 단계 타겟, 전환율 매우 높음. 6가지 유형:

1. **제품 카테고리**: 슬리핑백 vs 이불
2. **발달 단계 전환**: 속싸개 vs 슬리핑백 (뒤집기 시점)
3. **소재 간 비교**: 메쉬 vs 밤부 vs 트리플 (우리 라인업 내)
4. **원산지**: 국내산 vs 수입산
5. **브랜드 간**: A vs B vs 썬데이허그 (파워블로거 의뢰 전용)
6. **가격대**: 3만원대 vs 5만원대

**핵심 원칙**: 공정해 보이되 결론은 우리로. 비교표 + 실사진 + 구체적 수치 3박자 필수.

### STEP 7 — KPI + 성과 점검 체계 (마케팅 플래너 병합)

|------|---------|---------|

| 체험단 블로그 발행 | 15개 | 3-7-5개 |
| 배포형 발행 | 10~14건 | 4-6-4건 |
| 인스타 릴스 | 24회 | 8-8-8회 |
| 공동구매 | 1회 | 2~3월차 |
| 상위 노출 키워드 | 3개+ | 누적 |
| UGC 해시태그 | 50개+ | 누적 |

**월말 성과 점검 체크리스트**:
```
□ 네이버: 집중 키워드 3개 순위 확인 (목표: 20위 이내)
□ 스마트스토어: 블로그/배포형 유입 경로 비율
□ 인스타: 릴스 저장율 + 팔로워 증가
□ 맘카페: 후기 글 저장수 + 댓글 반응
□ 배포형: 건당 키워드 순위 변동 → ROI 판단
□ 다음달: 효과 좋은 채널 확대 / 비효율 채널 교체
```

### 필수 탭 구조 (8개 — 통합)

1. **전체 개요** — 기간/테마/KPI 서머리 + 이벤트 캘린더
2. **월별 상세** — 각 월 탭: 프로모션 카드 + 주차별 액션 (접기/펼치기)
3. **콘텐츠 전략** — 키워드 우선순위 버킷 + 콘텐츠 공식 + 배분 비율
4. **비교 콘텐츠** — 6가지 비교 유형 + 작성 원칙
5. **채널별 전략** — 채널 카드: 역할/수량/핵심 메시지
6. **핵심 키워드** — 검색 의도별 분류 + 월별 집중 키워드
7. **KPI 대시보드** — 목표 수치 + 월별 세부 KPI 표 + 예산
8. **성과 점검** — 월말 체크리스트 + 마일스톤 + SOP 스킬화

## 빠른 실행 체크리스트

```
□ 0. ★ 사전 기획 대화 (4가지 핵심 항목 확인 + 기획 요약 + 동의)
□ 1. 웹사이트 크롤링 (제품 + 현재 프로모션)
□ 2. 프로모션 기간 시즌 이벤트 매칭
□ 3. 월별 테마 + 프로모션 유형 배치
□ 4. 콘텐츠 발행 전략 + 비교 콘텐츠
□ 5. 채널별 주차 액션플랜 (쇼핑라이브 포함)
□ 6. KPI + 성과 점검 체계
□ 7. HTML 리포트 생성
□ 8. 사용자 피드백 반영 후 수정
```

## 재사용 주의사항

- 시즌이 바뀌면 STEP 1부터 새로 실행
- 같은 시즌이라도 3개월 이상 경과 시 웹사이트 재크롤링
- 프로모션 유형 연속 2회 이상 반복 지양
- HTML 리포트는 항상 현재 날짜 기준 일정 계산
- 맘카페 프로모션은 **브랜드 계정 직접 홍보 금지** 원칙 유지
- 3개월 완료 후 반드시 SOP 스킬화 체크리스트 실행
