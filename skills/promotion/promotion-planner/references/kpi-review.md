# KPI + 성과 점검 가이드

## 3개월 기준 기본 KPI

| 지표 | 3개월 목표 | 월별 배분 |
|------|---------|---------|
| 브랜드 블로그 발행 | 10개 | 3-4-3개 |
| 체험단 블로그 발행 | 15개 | 3-7-5개 |
| 배포형 발행 | 10~14건 | 4-6-4건 |
| 맘카페 체험단 후기 | 20개 | 5-10-5개 |
| 인스타 릴스 | 24회 | 8-8-8회 |
| 공동구매 | 1회 | 2~3개월차 |
| 상위 노출 키워드 | 3개+ | 누적 |
| 인스타 UGC 해시태그 | 50개+ | 누적 |

---

## 마일스톤 기준

| 월차 | 성공 기준 |
|-----|---------|
| 1월차 | 채널 셋업 완료 / 체험단 1기 발송 / 블로그 3개+ / 배포형 2건 |
| 2월차 | 블로그 누적 8개+ / 배포형 6건 / 맘카페 저장 30+ / 캠페인 론칭 |
| 3월차 | 공동구매 완료 / UGC 자생 흐름 / 배포형 10건+ 누적 / SOP 완성 |

---

## 월말 성과 점검 체크리스트

```
□ 네이버: 집중 키워드 3개 순위 확인 (목표: 20위 이내)
□ 네이버: 브랜드 블로그 월간 방문자 수 확인
□ 스마트스토어: 블로그/배포형 유입 경로 비율 확인
□ 인스타: 릴스 저장율 + 팔로워 증가 확인
□ 맘카페: 후기 글 저장수 + 댓글 반응 확인
□ 배포형: 건당 키워드 순위 변동 분석 → ROI 판단
□ 다음달: 효과 좋은 채널 비중 확대 / 비효율 채널 교체
```

---

## ROI 측정 방법

### 배포형 블로그 ROI
1. 발행 전: 해당 키워드 현재 순위 기록
2. 발행 2주 후: 순위 재확인
3. 발행 1개월 후: 스마트스토어 유입 경로에서 해당 블로그 트래픽 확인
4. 효과 없는 채널 → 교체, 효과 좋은 채널 → 반복 의뢰

### 체험단 ROI
- 후기 건수 대비 검색 노출 변화
- 후기 글 저장수/공감수
- 스마트스토어 리뷰 연동 여부

### 프로모션 ROI
- 할인 기간 매출 vs 정상가 매출 비교
- 신규 회원 가입수
- 객단가 변화
- 재구매율 (프로모션 후 30일 이내)

---

## 3개월 후 SOP 스킬화 체크리스트

```
□ 잘 된 콘텐츠 포맷 목록화 (릴스 구성, 블로그 제목 패턴)
□ 체험단 모집 → 선발 → 발송 → 가이드 → 발행 → 관리 SOP 문서화
□ 파워블로거 우수 채널 목록 + 건당 효과 기록
□ 키워드별 배포형 ROI 기록표 작성
□ 다음 시즌/제품 즉시 적용 가능한 콘텐츠 템플릿 정리
□ 해시태그 세트 (채널별) 정리
□ 프로모션 유형별 효과 분석 (어떤 유형이 가장 전환 높았는지)
□ 맘카페 공동구매 제휴처 연락처 + 조건 기록
```
