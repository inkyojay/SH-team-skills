# 비교 콘텐츠 전략 가이드

## 비교 콘텐츠가 강력한 이유

1. 구매 결정 직전 단계 타겟 → 전환율 높음
2. "A vs B" 형식은 공유·저장 확산에 유리
3. 롱테일 비교 키워드가 경쟁 낮음
4. 썬데이허그가 프레임을 주도하면 결론 통제 가능

---

## 6가지 비교 유형

### 유형 1 — 제품 카테고리 비교
- **주제**: 슬리핑백 vs 이불
- **프레임**: "아이 수면 안전, 어떤 게 나을까요?"
- **결론 유도**: "이불은 걷어찰 수 있지만 슬리핑백은 못 벗어나요"
- **채널**: 배포형 블로그 + 맘카페 정보글

### 유형 2 — 발달 단계 전환 비교
- **주제**: 속싸개 vs 슬리핑백
- **프레임**: "뒤집기 전후, 무엇을 입혀야 할까요?"
- **결론 유도**: "뒤집기 후 속싸개는 위험할 수 있어요"
- **채널**: 배포형 1순위 + 블로그 + 인스타 카드뉴스

### 유형 3 — 소재 간 비교 (우리 라인업 내)
- **주제**: 메쉬 vs 밤부 vs 트리플밤부
- **프레임**: "어떤 소재가 우리 아기에게 맞을까요?"
- **결론 유도**: "계절마다 맞는 소재가 달라요. 썬데이허그엔 다 있어요"
- **채널**: 블로그 심층글 + 스마트스토어 상세페이지

### 유형 4 — 원산지 비교
- **주제**: 국내산 vs 수입산
- **결론 유도**: "국내산은 소재 확인과 A/S가 다릅니다"
- **채널**: 배포형 블로그 + 맘카페 신뢰 글

### 유형 5 — 브랜드 간 비교
- **주제**: A vs B vs 썬데이허그
- **프레임**: "실제 써본 엄마가 비교한 후기"
- **결론 유도**: "국내산 + 3종 라인업 = 썬데이허그"
- **채널**: 파워블로거 의뢰 (브랜드 직접 발행 금지)

### 유형 6 — 가격대 비교
- **주제**: 3만원대 vs 5만원대
- **결론 유도**: "아기 피부에 닿는 것, 조금 더 투자할 이유"
- **채널**: 배포형 블로그 + 스레드

## 작성 원칙

### 공정성 유지
```
✓ 경쟁 제품 장점도 인정: "다른 브랜드도 소재는 좋아요. 다만..."
✓ 상황별 추천으로 중립 포지션
✓ 우리 제품이 강한 항목을 비교표 마지막에 배치
✗ "우리 제품이 최고" 단정 금지
✗ 경쟁 제품 깎아내리기 금지
```

### 필수 구성 요소
```
□ 비교 표 (소재/가격/원산지/사이즈/특징)
□ 실제 사용 사진 (자연광, 착용 상태)
□ 구체적 수치 (혼방 비율, cm, 가격)
□ 상황별 추천 정리
□ 자연스러운 결론 + 스마트스토어 링크
```

### 채널별 형태
| 채널 | 형태 | 분량 |
|------|------|------|
| 네이버 블로그 | 비교 심층 글 | 2,000자+ + 비교표 |
| 배포형 | 파워블로거 배포 | 1,500자+ |
| 인스타 | 카드뉴스 5~8장 | 장당 50자 이내 |
| 맘카페 | 정보형 후기 | 1,000자+ + 사진 3장 |
| 스레드 | 대화형 답변 | 200자 이내 |
