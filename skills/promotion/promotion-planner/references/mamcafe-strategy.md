# 맘카페 바이럴 전략 가이드

## 핵심 원칙

**브랜드 계정으로 직접 참여 절대 금지.**
맘카페는 광고가 들키는 순간 역효과 + 계정 정지 위험.

---

## 4가지 바이럴 전략

### 전략 1 — 체험단 진성 후기

**선발 기준**: 맘카페 활동 이력 5글+ / 정회원 이상 / 해당 연령대 아이

**가이드 키트**:
```
□ 후기 주제 3가지 중 선택
  - "뒤집기 시작하고 슬리핑백으로 바꿨어요"
  - "태열 아기 여름 수면 고민 해결한 제품"
  - "수면교육 3개월째 — 슬리핑백 효과 진짜?"
□ 필수 키워드 1~2개 자연 포함
□ 사진: 자연광 / 아기 착용 / 소재 클로즈업
□ 분량: 800자+ / 사진 3장+
□ 말미: "체험단 제공받아 작성했습니다" (법적 의무)
```

### 전략 2 — 질문글 댓글 참여

**진입 키워드**: "수면조끼 추천", "슬리핑백 뭐 써요", "이불 차는 아기"

**댓글 방식**:
```
✗ "썬데이허그 써보세요! 좋아요!"
✓ "저도 비슷한 고민으로 한참 찾다가 슬리핑백으로 바꿨어요.
   뒤집기 시작하고 나서 썼는데 확실히 달라지더라고요."
```
목표: 주 3~5개 글, 체험단 1인당 주 1~2개

### 전략 3 — 정보글 바이럴 (저장수 특화)

**저장 유도 글 공식**:
```
제목: [유용한 정보] + "(저장필수)" 또는 "(정리해봤어요)"
예: "실내온도별 아기 착용 가이드 총정리 (저장필수)"
본문: 공감 도입 → 정보 테이블 → 실제 팁 → 제품 1줄 자연 언급
분량: 1,000자+ / 표 + 이미지 필수
목표: 저장수 50+
```

### 전략 4 — 공동구매 제휴

**타이밍**: 시즌 피크 1~2개월 전 준비, 피크 직후 오픈
**조건**: 카페 전용 할인코드 10~15% / 번들 특가 / D-day 카운트다운
**컨택**: 카페 공지 "공동구매 신청" → 운영자 쪽지/이메일

## 주요 타겟 카페

| 우선순위 | 카페명 | 공략 게시판 |
|--------|-------|-----------|
| 최우선 | 맘스홀릭베이비 | 수면·육아용품 |
| 최우선 | 미즈넷 | 제품 후기 |
| 2순위 | 아기랑 | 월령별 게시판 |
| 2순위 | 지역 맘카페 | 공구 게시판 |

## 주의사항

```
✓ 아기 스토리를 주인공으로 / 자연광 사진 필수
✗ 브랜드 계정 직접 홍보 / 동일 글에 다중 계정 댓글 / 권유형 직접 표현
```
