SUNDAYHUG SKILL

/promo-html 사용 가이드

기획부터 14개 채널 HTML 디자인 생성까지 원스톱 프로모션 제작

빠른 시작

Claude Code에서 아래 명령어를 입력하세요:

$ /promo-html

기존 기획서(plan.json)가 있으면 자동으로 찾아 로드하고, 없으면 대화형으로 기획을 시작합니다.

7단계 워크플로우

스킬은 7개 Phase를 순서대로 진행하며, 각 단계마다 사용자 확인을 거칩니다.

1
인터랙티브

기획서 로딩 또는 신규 기획

기존 plan.json 검색 → 있으면 로드, 없으면 대화형으로 프로모션 기획 (프로모션명, 컨셉, 채널, 혜택, 제품 등)

2
인터랙티브

채널별 디자인 목록 확정

선택한 채널에 해당하는 14개 포맷 중 생성할 디자인 목록을 테이블로 확인

3
인터랙티브

채널별 카피 및 디자인 방향

채널 특성에 맞는 헤드라인/서브카피/뱃지 생성 → 사용자 확인

4
병렬 처리

HTML 병렬 생성

채널별 에이전트가 병렬로 HTML 파일 생성 — 레퍼런스 이미지 참고, 브랜드 디자인 시스템 적용

5
인터랙티브

검토 및 수정

생성된 HTML 목록 확인 → 사용자 피드백 → 즉시 수정 반영

6
자동

PNG 변환 + .pen 생성

Playwright로 HTML→PNG 배치 변환, Pencil MCP로 .pen 파일 생성

7
자동

완료 리포트

전체 산출물 현황 테이블 + 파일 경로 안내

채널 자동 매핑

plan.json에서 "인스타그램"을 선택하면 스토리/피드/재입고/체험단 4개 포맷이 자동으로 포함됩니다. 원하지 않는 포맷은 Phase 2에서 제외할 수 있습니다.

출력 폴더 구조

{{WORKSPACE_DIR}}/output/{slug}/
├── plan.json ← 기획 데이터
├── html/ ← HTML 소스 파일
│ ├── 01-insta-story.html
│ ├── 02-insta-event.html
│ ├── ...
│ └── 14-live-page.html
├── png/ ← PNG 렌더링 결과 (2x)
│ ├── 01-insta-story.png
│ └── ...
└── pen/ ← Pencil 에디터 파일
  ├── 01-insta-story.pen
  └── ...

PNG 수동 변환 (필요 시)

Phase 6에서 자동으로 실행되지만, HTML을 수정한 후 수동으로 다시 변환할 수도 있습니다.

사전 설치

$ pip install playwright
$ playwright install chromium

전체 배치 변환

$ python3 {{REPO_DIR}}/skills/promotion/promo-html/scripts/render_all.py \
  --input-dir "{{WORKSPACE_DIR}}/output/{slug}/html/" \
  --output-dir "{{WORKSPACE_DIR}}/output/{slug}/png/" \
  --scale 2

옵션

옵션기본값설명
--input-dir필수HTML 파일 폴더
--output-dir필수PNG 저장 폴더
--scale2.0해상도 배율 (2x = 레티나)
--concurrency3동시 렌더링 수

스킬 파일 구조

promo-html/
├── SKILL.md ← 메인 스킬 (7 Phase 워크플로우)
├── references/
│ ├── design-system.md ← 브랜드 컬러/타이포/컴포넌트
│ ├── channel-specs.md ← 14개 채널 사이즈/레이아웃
│ ├── variable-guide.md ← plan.json → HTML 변수 매핑
│ └── html-generation-rules.md ← HTML 작성 규칙/금지사항
├── scripts/
│ └── render_all.py ← HTML→PNG 배치 변환
└── guide.html ← 이 가이드 문서

심볼릭 링크: ~/.claude/commands/promo-html.md → SKILL.md

팁

기존 promotion-plan과 호환

/promotion-plan으로 만든 plan.json을 그대로 사용할 수 있습니다. JSON 스키마가 동일합니다.

HTML 직접 수정

Phase 5에서 피드백을 주면 Claude가 HTML을 직접 수정합니다. 또는 VS Code 등에서 직접 열어 수정한 뒤 PNG만 재변환할 수도 있습니다.

부분 재생성

"인스타 스토리만 다시 만들어줘"처럼 특정 채널만 재생성 요청할 수 있습니다.

Playwright 필요

PNG 변환에는 Playwright가 필요합니다. 미설치 시 pip install playwright && playwright install chromium을 실행하세요.
