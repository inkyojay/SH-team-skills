# 키워드 분석 가이드

## 네이버 API JSON 데이터 해석

### 등급 분류 기준

| 등급 | 총 검색량 | 경쟁도 | 전략 |
|------|-----------|--------|------|
| 🏆 골든 | 1,000~10,000 | 낮음 | 메인 타겟 키워드 |
| 📈 성장 | 1,000~10,000 | 중간 | 소제목/본문 배치 |
| 🔥 빅 | 10,000+ | 높음 | 참고/연관 배치 (직접 경쟁 어려움) |
| 🔹 니치 | 100~1,000 | 낮음 | 롱테일 보조 키워드 |

### 블로그 글 1편의 키워드 선택 원칙

**메인 키워드 선정 (1개):**
- 골든 > 성장 키워드 중 글 주제와 가장 일치하는 것
- 너무 짧은 단어(1음절)보다 2~3어절 구체 키워드 선호
- 예: "슬리핑백"(빅, 경쟁 높음)보다 "신생아 슬리핑백 추천"(골든) 선택

**서브 키워드 선정 (3~5개):**
- 메인 키워드 연관 키워드 중 선택
- 본문 소제목 or 내용 흐름에 자연스럽게 포함 가능한 것
- 검색량 100~3,000 범위 권장

**LSI 키워드 (5~10개):**
- 같은 주제 주변 단어들 (검색량 무관)
- 자연스러운 문맥에서 등장
- 예: 슬리핑백 글에서 "수면 온도", "신생아 체온", "잠자리 준비"

## 키워드 선택 판단 기준

### 좋은 타겟 키워드의 조건
1. **검색 의도 명확**: "슬리핑백 추천" (구매형) vs "슬리핑백이란" (정보형) 구분
2. **적정 경쟁도**: 낮음~중간 (높음은 대형 사이트가 이미 점령)
3. **계절성 확인**: 시즌 키워드라면 시즌 전 2~4주 전 발행
4. **모바일 비율**: 70% 이상이면 가독성 최우선 구조 설계

### 피해야 할 키워드
- 검색량 < 100: 유입 효과 미미
- 경쟁도 높음 + 빅 키워드: 신생 블로그로는 상단 불가
- 트렌드성 키워드: 지속적 유입 어려움

## 키워드 배치 계획표 (글 작성 전 작성)

```
메인 키워드: 신생아 슬리핑백 추천
검색량: 월 3,200회 / 경쟁도: 낮음 / 등급: 골든

서브 키워드:
- 신생아 슬리핑백 사이즈 (검색량 1,100) → H2 소제목에 배치
- 슬리핑백 소재 (검색량 850) → 본문 중반 배치
- 아기 수면자루 (검색량 2,400) → 도입부 연관 언급

LSI 키워드:
- 신생아 수면, 아기 잠자리, TOG, 면 소재, 대나무 소재, 겨울 이불
→ 자연스럽게 본문에 배치 (위치 무관)

키워드 목표 밀도 계획:
- 총 글자수 목표: 2,500자
- 메인 키워드 목표: 15회 (밀도 약 2%)
- 서브 키워드 합계: 각 5~8회
```

## 경쟁 블로그 분석 체크리스트

web_search로 타겟 키워드 검색 후 상위 1~5위 블로그 분석:

### 분석 항목
- [ ] 제목 패턴 (숫자형? 질문형? 최신연도 포함?)
- [ ] 글 길이 (짧은 편 / 중간 / 긴 편)
- [ ] 소제목 수와 구성 순서
- [ ] 이미지 수와 유형 (제품사진 / 인포그래픽 / 직접 촬영)
- [ ] 브랜드 언급 여부
- [ ] 발행 날짜 (최근성 확인)

### 차별화 전략 도출
분석 후 다음 3가지 중 1개 이상 선택:
1. **더 자세하게**: 경쟁 글에 없는 정보 추가
2. **더 구조화**: 표, 체크리스트, 단계형 구성
3. **더 신뢰있게**: 실제 경험/데이터/전문가 인용 추가
