# 네이버 블로그 SEO 규칙 가이드

## 네이버 검색 알고리즘 이해

### C-Rank (콘텐츠 신뢰도)
네이버 블로그 상위 노출의 핵심 지표. 문서 자체의 신뢰도와 출처의 신뢰도를 평가.

**C-Rank 높이는 방법:**
- 특정 주제에 집중한 블로그 (전문성)
- 직접 작성한 오리지널 콘텐츠
- 이미지를 직접 찍거나 제작
- 꾸준한 발행 이력 (최소 주 2-3회)
- 댓글, 공감, 저장 수 (독자 반응)

**C-Rank 낮추는 것:**
- 복사/붙여넣기 텍스트
- 과도한 외부 링크
- 과도한 광고성 문구
- 모든 주제 무분별 작성

### DIA (Deep Intent Analysis)
사용자 검색 의도와 콘텐츠의 일치도 평가. 검색어의 진짜 의도에 맞는 글이 상위 노출.

**검색 의도 유형:**
| 유형 | 예시 | 글 방향 |
|------|------|---------|
| 정보형 | "슬리핑백 고르는 법" | 체크리스트, 비교, 가이드 |
| 구매형 | "슬리핑백 추천" | 제품 비교, 순위, 장단점 |
| 경험형 | "슬리핑백 후기" | 실사용 리뷰, 솔직한 평가 |
| 방법형 | "슬리핑백 세탁 방법" | 단계별 방법, 주의사항 |

## SEO 제목 작성법

### 클릭률 높은 제목 패턴
```
[숫자형]  신생아 슬리핑백 고르는 법 5가지
[질문형]  신생아 슬리핑백, 어떻게 골라야 할까요?
[최신형]  2025 신생아 슬리핑백 추천 TOP 5
[경험형]  신생아 슬리핑백 직접 써보고 솔직하게 알려드립니다
[해결형]  신생아 슬리핑백 고민, 이 글 하나로 해결!
[비교형]  신생아 슬리핑백 vs 속싸개, 뭐가 다를까?
```

### 제목 SEO 체크리스트
- [ ] 메인 키워드가 제목 앞 1/3에 위치
- [ ] 전체 길이 35자 이내
- [ ] 클릭 유도 단어 포함 (추천, 후기, 방법, 비교, 총정리)
- [ ] 특수문자 과용 금지 (| 하나 정도는 OK)

## 본문 SEO 최적화

### 키워드 배치 전략

**메인 키워드 배치 위치:**
1. 제목 (필수)
2. 첫 문단 100자 내 (필수)
3. 소제목(H2) 중 1~2개
4. 본문 중반부
5. 마지막 문단

**서브/연관 키워드 배치:**
- 소제목(H2, H3)에 자연스럽게 배치
- 본문 내 문맥에 맞게 자연스럽게 삽입
- 키워드 스터핑(억지 반복) 절대 금지

**키워드 밀도 계산:**
```
키워드 밀도 = (키워드 등장 횟수 / 총 단어 수) × 100
권장 범위: 1.5% ~ 2.5%
예: 2,500자 기준 메인 키워드 15~20회 등장이 적절
```

### 글 분량 기준
| 목적 | 최소 | 권장 | 상한 |
|------|------|------|------|
| 정보성 | 1,500자 | 2,500자 | 5,000자 |
| 리뷰형 | 1,200자 | 2,000자 | 4,000자 |
| 비교형 | 2,000자 | 3,000자 | 6,000자 |
| 방법형 | 1,000자 | 1,800자 | 3,000자 |

## 도입부 작성법 (첫 문단 200-300자)

**공식: 공감 → 문제 → 해결 예고 → 키워드**

```
예시:
"아기가 태어나고 처음으로 맞닥뜨리는 고민 중 하나가 
바로 '이불을 어떻게 덮어줄까?'예요. 
작은 손발을 꼼지락거리며 이불을 차버리는 신생아 때문에 
새벽마다 잠에서 깨신 분 계시죠?

신생아 슬리핑백이 그 고민을 해결해 드릴 수 있어요.
오늘은 슬리핑백을 처음 고르는 분들을 위해
사이즈 선택부터 소재 체크포인트까지 알려드릴게요."
```

## 소제목(H2) 구성 전략

### 정보성 글 소제목 패턴
```
H2: [키워드]란? (개념 정의)
H2: [키워드] 고르는 법 / 선택 기준
H2: [키워드] 추천 제품 / 인기 아이템
H2: [키워드] 사용 방법 / 활용 팁
H2: [키워드] 자주 묻는 질문 (FAQ)
H2: 마무리 / 요약
```

### 리뷰형 글 소제목 패턴
```
H2: [제품명] 스펙/상세 정보
H2: [제품명] 직접 사용해보니
H2: 좋았던 점 3가지
H2: 아쉬운 점 (솔직 후기)
H2: 이런 분께 추천합니다
H2: 최종 평가 + 별점
```

### 비교형 글 소제목 패턴
```
H2: [A] vs [B] 핵심 차이
H2: [A]의 장단점
H2: [B]의 장단점
H2: 상황별 추천 (케이스별)
H2: 가격 비교
H2: 최종 추천
```

## 이미지 SEO 최적화

### 이미지 개수 및 배치
- 최소 3개, 권장 5~7개
- 첫 번째 이미지: 도입부 직후 (대표 이미지)
- 중간 이미지: 소제목 1~2개마다 1개
- 마지막 이미지: 마무리 전

### 이미지 파일명 규칙
```
✅ 좋은 예: 신생아-슬리핑백-추천-2025.jpg
❌ 나쁜 예: IMG_0123.jpg, image1.jpg
```

### alt 텍스트 작성법
```
<img src="..." alt="신생아 슬리핑백 사이즈 비교 - 0-6개월용과 6-12개월용">
```
- 이미지 내용을 키워드 포함하여 구체적으로 설명
- 40자 이내 권장


## 발행 최적화

### 발행 시간 (트래픽 피크)
- **평일**: 오전 7~9시, 오후 7~10시
- **주말**: 오전 9~11시
- **피해야 할 시간**: 새벽 2~5시, 평일 낮 12~1시

### 발행 후 할 일 (첫 1시간)
1. 공감/하트 자가 클릭
2. 다른 블로그 댓글 활동 (자연스러운 유입 유도)
3. 카카오채널/인스타 스토리에 링크 공유
4. 해당 키워드로 검색 후 내 글 클릭 (CTR 신호)

### 주의사항
- 발행 직후 수정하면 SEO 점수 초기화 위험 → 신중하게 최종 검토 후 발행
- 짧은 기간 대량 발행보다 꾸준한 발행이 C-Rank에 유리
