# PHASE 8 — 제품 마케팅 소구점 설계 가이드

## 목적

PHASE 1~7의 모든 데이터를 종합하여, 마케팅에서 실제로 사용할 소구점(Appeal Points)을 체계적으로 설계한다. 상세페이지, 광고 카피, SNS 콘텐츠, 체험단 가이드에 바로 활용 가능한 형태로 산출.

## 소구점 설계 4계층

### 1층: 기능적 소구점 (Functional)
"이 제품이 뭘 해주는가"

데이터 소스: PHASE 2.5 경쟁사 분석 + PHASE 2.6 VOC 분석 + PHASE 3 경쟁사 약점

| # | 소구점 | 근거 데이터 | 예시 카피 |
|---|--------|-----------|---------|
| F1 | 냉감 효과 | Q-MAX 수치, 소재 스펙 | "닿는 순간 -3°C 체감" |
| F2 | 보풀 방지 | 경쟁사 불만 1위 해결 | "100회 세탁해도 보풀 없는" |
| F3 | 논슬립 | 경쟁사 불만 상위 | "밤새 밀리지 않는 안전 설계" |
| ... | ... | ... | ... |

추출 방법:
- 경쟁사 리뷰 불만 TOP 5 → 우리가 해결한 것 = 소구점
- 우리 제품만의 기술/소재 차별점
- 스펙 수치 (Q-MAX, 중량, 사이즈 등)

### 2층: 감성적 소구점 (Emotional)
"이 제품을 쓰면 어떤 기분인가"

데이터 소스: PHASE 2.6 맘카페 목소리 + PHASE 1 타겟 분석

| # | 소구점 | 근거 | 예시 카피 |
|---|--------|------|---------|
| E1 | 아기 꿀잠 안심 | 맘카페 "잘 자요" 후기 패턴 | "오늘 밤도 꿀잠, 엄마도 푹 자요" |
| E2 | 엄마의 현명한 선택 | 추천 요청 글 패턴 | "까다로운 엄마가 선택한" |
| E3 | 국내산 신뢰 | VOC 안전성 우려 패턴 | "우리 아이 피부에 닿으니까, 국내산" |

추출 방법:
- 맘카페 만족 키워드 → 감성 카피 소재
- 구매 전 고민(지식인) → 해소해주는 메시지
- 타겟 페르소나의 일상 공감 포인트

### 3층: 사회적 소구점 (Social Proof)
"다른 사람들도 선택했다"

| # | 소구점 | 근거 | 예시 카피 |
|---|--------|------|---------|
| S1 | 맘카페 인기 | 카페 언급 빈도 | "맘카페에서 입소문 난" |
| S2 | 리뷰 볼륨 | 블로그 리뷰 수 | "1,000명의 엄마가 선택한" |
| S3 | 브랜드 신뢰 | 기존 슬리핑백 판매 실적 | "슬리핑백 베스트셀러 썬데이허그의" |

### 4층: 긴급성/시기 소구점 (Urgency)
"지금 사야 하는 이유"

| # | 소구점 | 근거 | 예시 카피 |
|---|--------|------|---------|
| U1 | 시즌 한정 | 시즌 트렌드 데이터 | "올 여름 한정 런칭 특가" |
| U2 | 얼리버드 | 프로모션 전략 | "사전예약 고객만 20% 할인" |
| U3 | 세트 한정 | 번들 전략 | "세트 구매 시 베개 무료 증정" |

## 소구점 → 콘텐츠 매핑

각 소구점이 어떤 채널/콘텐츠에 사용되는지 매핑:

| 소구점 | 상세페이지 | 블로그 | 인스타 | 맘카페 | 광고 |
|--------|----------|--------|--------|--------|------|
| F1 냉감 | 메인 배너 | 비교 콘텐츠 | 릴스 | 추천글 | 메인 카피 |
| E1 꿀잠 | 감성 섹션 | 스토리텔링 | 피드 | 후기 | 서브 카피 |
| S1 입소문 | 리뷰 섹션 | 후기 모음 | 스토리 | 자연 언급 | CTA |
| U1 한정 | 배너 | — | 스토리 | — | CTA |

## 상세페이지 스토리라인 제안

소구점을 기반으로 상세페이지 구성 순서 제안:

1. **훅 (E1)**: 감성 이미지 + "아기 꿀잠의 비밀"
2. **문제 제기 (VOC)**: "여름밤, 아기가 자꾸 깨나요?"
3. **해결책 (F1~F3)**: 제품 기능 소개 + 스펙 수치
4. **신뢰 (S1~S3)**: 리뷰/인증/브랜드 스토리
5. **구매 이유 (U1~U3)**: 프로모션 + CTA

## 리포트 반영

HTML 리포트의 "마케팅 소구점" 탭:
- 4계층 소구점 카드 (기능/감성/사회적/긴급성)
- 각 소구점별 근거 데이터 출처 표시
- 소구점 → 채널 매핑 테이블
- 상세페이지 스토리라인 플로우 차트
- 예시 카피 목록 (복사해서 바로 사용 가능)
