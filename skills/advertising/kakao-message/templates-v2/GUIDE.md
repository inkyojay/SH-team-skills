# 카카오 메시지 디자인 템플릿 시스템

> 썬데이허그 브랜드의 카카오톡 비즈메시지 + 알림톡 디자인 템플릿
> HTML → 이미지 변환 용도 · JS/애니메이션 없음

## 개요

| 항목 | 내용 |
|------|------|
| 메시지 타입 | 8종 |
| 디자인 베리에이션 | 24개 |
| 캠페인 팔레트 | 12개 |
| 변환 방식 | HTML → PNG (브라우저 캡처) |

## 파일 구조

```
kakao-messages/
├── _base-styles.css          ← 카카오 전용 CSS
├── _palettes.css             ← 12개 캠페인 팔레트
├── _guide-chrome.css         ← 가이드 UI 스타일
├── GUIDE.md                  ← 이 문서
├── template-guide.html       ← 비주얼 가이드 (타입 탭 + 팔레트 전환)
└── types/                    ← 메시지 타입별 HTML 템플릿
    ├── 01-image/             ← 이미지형 (800×400)
    ├── 02-wide-image/        ← 와이드 이미지형 (800×600)
    ├── 03-wide-list/         ← 와이드 리스트형 (헤더+리스트)
    ├── 04-carousel-feed/     ← 캐러셀 피드형 (800×600/카드)
    ├── 05-carousel-commerce/ ← 캐러셀 커머스형 (800×800/카드)
    ├── 06-commerce/          ← 커머스형 (800×600)
    ├── 07-alimtalk-image/    ← 이미지 알림톡 (800×400)
    └── 08-alimtalk-itemlist/ ← 아이템리스트 알림톡
```

|---|--------|------------|------|-----------|------|

## 베리에이션 전략

모든 타입에 일관된 레이블 사용:

- **A — Product / Standard / Order**: 제품 사진 중심, 가격/할인 강조
- **B — Lifestyle / Dual / Curated / Info**: 감성 이미지 또는 2개 제품 비교
- **C — Event / Sale / Story / Triple**: 이벤트/할인/스토리텔링 또는 3개 제품 리스트
- **D — Option**: 옵션 A/B 비교 + 가격표 + 쿠폰 (캐러셀 전용)
- **E — Bundle**: 세트 상품 카드 (캐러셀 전용)

### 캐러셀 카드 구성

캐러셀 타입(04, 05)은 **커버 카드 + 후속 카드** 세트로 구성:
- **커버 카드**: 프로모션 소개 (라이프스타일 사진 또는 그라데이션 배경)
- **후속 카드**: 개별 제품/비교/리스트 등 다양한 레이아웃

| 레이아웃 | 설명 | 제품 수 |
|---------|------|---------|
| Single | 단일 제품 + 가격 | 1 |
| Dual | 2개 제품 나란히 비교 | 2 |
| Triple | 넘버링 리스트 + 썸네일 | 3 |
| Option | A/B 옵션 비교 + 가격표 | 2 |
| Bundle | 세트 구성 + 합산 가격 | 2~3 |

## 사용법

### 1. 템플릿 선택

목적에 맞는 타입과 베리에이션을 선택합니다.

### 2. 이미지 교체

`src="..."` 경로를 실제 제품 이미지로 교체합니다.

### 3. 텍스트 수정

`{{placeholder}}` 패턴의 텍스트를 실제 콘텐츠로 교체합니다.

### 4. 팔레트 적용

부모 요소에 `data-palette="팔레트명"` 속성을 추가합니다.

```html
<div data-palette="warm-spring">
  <!-- 템플릿 HTML -->
</div>
```

### 5. 이미지 변환

HTML 파일을 브라우저에서 열고, 해당 프레임을 캡처합니다.
- 정확한 픽셀 크기로 캡처 (800×400, 800×600, 800×800)
- `object-fit: cover`로 이미지 비율 자동 조정

## 제약사항

- **JS/애니메이션 완전 금지** — HTML → 이미지 변환 용도
- **카카오 규격 정확히 준수** — 800×400, 800×600, 800×800
- **`object-fit: cover`** 필수 — 이미지 비율 강제
- **midnight-luxe 팔레트** — 다크 배경 별도 오버라이드 적용됨

## 검증 체크리스트

1. `template-guide.html`을 브라우저에서 열기 (file:// 호환)
2. 8개 타입 탭 전환 확인
3. 12개 팔레트 전환 시 모든 베리에이션에 색상 적용 확인
4. 각 프레임이 정확한 카카오 규격 크기인지 개발자도구로 확인
5. 이미지 렌더링 시 잘림/여백 없는지 확인
