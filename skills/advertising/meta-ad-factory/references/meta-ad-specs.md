# 메타(Facebook/Instagram) 광고 규격 가이드

## 이미지 광고 사이즈

| 포맷 | 비율 | 픽셀 | 용도 | 우선순위 |
|------|------|------|------|----------|
| 피드 정사각 | 1:1 | 1080×1080 | FB/IG 피드 공통 | ★★★ |
| IG 피드 세로 | 4:5 | 1080×1350 | IG 피드 최적 (면적 최대) | ★★★ |

## 텍스트 규격

### Primary Text (본문)
- 권장: 125자 이내 (더 길면 "더 보기"로 잘림)
- 최대: 테스트 목적으로 최대 3줄까지
- 이모지 사용 가능 (첫 줄에 주의 환기용)
- 줄바꿈으로 가독성 확보

### Headline (헤드라인)
- 권장: 40자 이내
- 이미지 바로 아래 굵은 텍스트
- 핵심 메시지 / 혜택 / CTA

### Description (설명)
- 권장: 30자 이내
- 헤드라인 바로 아래 회색 텍스트
- 보조 정보 (배송, 보증 등)

### CTA 버튼 옵션
- 지금 구매하기 (Shop Now)
- 더 알아보기 (Learn More)
- 가입하기 (Sign Up)
- 지금 주문하기 (Order Now)

## 이미지 내 텍스트 규칙

- 메타 권장: 이미지 면적의 20% 이하에 텍스트
- 실제로는 20% 초과해도 게재되지만 도달률 감소
- **베스트 프랙티스**: 핵심 한 줄 + 가격/할인만 이미지에 넣고, 나머지는 카피 영역 활용

## 파일 규격

- 포맷: JPG, PNG (PNG 권장 — 텍스트 선명도)
- 최대 파일 크기: 30MB
- 권장 해상도: 최소 1080px 너비

## 광고 유형별 권장 레이아웃

| 광고 목적 | 권장 레이아웃 | 권장 사이즈 |
|-----------|-------------|------------|
| 브랜드 인지도 | hero-image, split | 1:1, 4:5 |
| 트래픽 | benefit-stack, split | 1:1 |
| 전환 (구매) | urgency, social-proof | 4:5 |
| 리타겟팅 | urgency, hero-image | 1:1, 9:16 |
| 스토리 광고 | hero-image, urgency | 9:16 |

## 캐러셀 광고

- 카드 수: 2~10장
- 각 카드: 1080×1080 (1:1) 권장
- 각 카드에 개별 링크 가능
- 활용: 제품 라인업, 기능 순차 소개, 스토리텔링
