# 라이브 커머스 페이지 빌더

라이브 방송용 상세 페이지를 모듈형 섹션으로 조립합니다.

## 구조

- `sections/` — 라이브 방송 전용 섹션 템플릿
- `GUIDE.md` — 섹션 조립 가이드
- `_base-styles.css` — 공통 스타일
- `_palettes.css` — 컬러 팔레트
- `template-guide.html` — 비주얼 가이드

## 사용법

1. `sections/` 에서 라이브 방송에 맞는 섹션 선택
2. 가격/혜택/타임딜 정보 삽입
3. 팔레트 적용 후 내보내기

## 디자인 시스템

공유 팔레트: `skills/advertising/_design-system/` 참조
