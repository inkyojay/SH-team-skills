# STEP 2 — USP 분석 가이드

## 6가지 USP 차원 체크리스트

각 차원에서 해당 제품의 강점을 찾는다.

### 1. 기능적 차별점
- 소재의 물리적 특성 (통기성, 보온성, 흡습성)
- 독자 구조 (투웨이 지퍼, 낙상방지 설계)
- 착용/탈의 편의성

### 2. 안전성/신뢰
- 원산지 (100% 국내산 = 강력 소구점)
- 소재 안전성 (무형광, 천연 소재)
- 인증 여부

### 3. 편의성
- 야간 기저귀 교체 편의 (아래서 여는 지퍼)
- 탈의 방지 설계 (M/L 사이즈)
- 세탁 용이성

### 4. 성장 대응
- 사이즈 범위 (S: ~9개월 / M: ~36개월 / L: ~60개월)
- 사이즈별 기능 차별화
- 장기 사용 가능 여부

### 5. 감성적 가치
- "입는 이불" — 부모 불안 해소
- 수면 루틴의 신호로 기능
- 가족 모두 편안한 밤 메시지

### 6. 시즌/온도 대응
- 적합 실내 온도 범위
- 계절별 착용 조합 (레이어링 가이드)
- 타 제품과 조합 가능성

## 헤드라인 카피 작성 공식

5가지 감성 유형으로 각 1개 이상 작성:

| 유형 | 공식 | 예시 |
|------|------|------|
| 감성/공감 | [고통 인식] + [안심 메시지] | "이불 걷어찬 밤, 이젠 걱정 마세요" |
| 기능/효익 | [소재/기능] + [결과] | "코튼 97% 에어메쉬 — 태열 아기도 보송하게" |
| 신뢰/원산지 | [원산지] + [소재] + [브랜드] | "100% 국내산, 직접 만든 슬리핑백" |
| 루틴/행동 | [제품 행동] + [결과 연상] | "슬리핑백 입히면 아기도 알아요 — 잘 시간" |
| 시즌/상황 | [상황] + [해결] | "여름 에어컨 바람 걱정 끝" |

## 타겟 구매 시나리오 4가지

썬데이허그 공통 시나리오 (제품별로 추가/수정):

1. **뒤집기 전환 시점** — 속싸개 졸업하고 슬리핑백 첫 구매
2. **시즌 특화** — 여름/겨울 수면 문제 해결
3. **수면 교육 시작** — 루틴 형성 도구로 구매
4. **선물 구매** — 출산 선물, 어린이날 선물 탐색
