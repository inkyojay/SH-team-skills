# 배포형 블로그 전략 가이드

## 왜 배포형이 필요한가

브랜드 블로그는 신규 도메인이라 네이버 신뢰도가 낮다.
파워블로거 채널은 이미 최적화되어 동일 콘텐츠도 더 빠르게, 더 높이 노출된다.

---

## 배포형 4가지 타입

### Type 1 — 파워블로거 원고 의뢰 (유료)
- 비용: 건당 8~15만원
- 플랫폼: 픽업엔, 애드픽, 직접 컨택
- 적합: 경쟁 중간~높음 키워드, 빠른 노출

### Type 2 — 자체 체험단 블로그 (저비용)
- 비용: 제품 원가 (2~5만원)
- 적합: 경쟁 낮은 키워드, 진성 후기

### Type 3 — 비교 특화 배포
- 비교 원고를 파워블로거에 의뢰
- 적합: "슬리핑백 브랜드 비교", "속싸개 vs 슬리핑백"

### Type 4 — 육아 미디어 기고
- 베이비뉴스, 맘앤앙팡, 월간유아 등
- 효과: 네이버 뉴스 탭 노출 + 브랜드 공신력

---

## 파워블로거 의뢰 SOP

### 원고 필수 포함 요소
```
□ 타겟 키워드 1~2개 (제목 + 본문 2회+)
□ 구체적 수치 (온도, 사이즈, 가격)
□ 비교 표 또는 가이드 표
□ 스마트스토어 링크 1~2곳
□ 실제 사용 사진
□ 분량: 1,500자+
```

### 섭외 기준
- 이웃 수 1,000+ / 최근 발행 활발 / 네이버 최적화

### 발행 후 모니터링
- 즉시: 키워드 검색 → 위치 확인
- 2주 후: 순위 변동 확인
- 목표: 10~20위 이내
