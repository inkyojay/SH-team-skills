# 프로모션 유형 가이드

## 10가지 프로모션 유형 상세

### 1. 시즌 오픈 세일
- **목적**: 신시즌 제품 인지도 + 초기 구매 유도
- **할인율**: 10~20%
- **기간**: 7~14일
- **대상**: 시즌 신상품 + 교체 수요 제품
- **채널 포커스**: 스토어 메인 배너 + 인스타 릴스 + 네이버 블로그
- **KPI**: 신상품 판매량, 페이지 유입수

### 2. 타임특가
- **목적**: 긴급성으로 즉시 구매 전환
- **할인율**: 20~35%
- **기간**: 24~72시간
- **대상**: 재고 조절 필요 제품 or 주력 상품 1~2종
- **채널 포커스**: 스토어 팝업 + 인스타 스토리 + 카카오 알림톡
- **KPI**: 전환율, 주문 건수

### 3. 번들 세트 특가
- **목적**: 객단가 상승 + 선물 수요 공략
- **할인율**: 15~30% (세트 기준)
- **기간**: 이벤트 전후 2~3주
- **구성 예시**: 슬리핑백 + 속싸개 + 바디수트 세트
- **채널 포커스**: 스토어 기획전 + 맘카페 추천글 + 배포형
- **KPI**: 세트 판매량, 객단가

### 4. 체험단
- **목적**: UGC 후기 볼륨 확보 → SEO + 신뢰도
- **비용**: 제품 원가 (무상 제공)
- **기간**: 모집 1주 + 사용 2주 + 후기 작성 1주
- **규모**: 회당 10~20명
- **플랫폼**: 네이버 블로그 체험단 (레뷰/미블 등)
- **KPI**: 후기 건수, 검색 노출 순위 변화

### 5. 맘카페 공동구매
- **목적**: 바이럴 + 대량 전환 동시 달성
- **할인율**: 20~30% (공구 전용가)
- **기간**: 3~7일
- **전제조건**: 리뷰 볼륨 10건+ 확보 후 진행
- **채널**: 맘카페 운영자 제휴
- **KPI**: 공구 판매량, 신규 고객 비율

### 6. UGC 캠페인 (해시태그 이벤트)
- **목적**: 고객 참여형 콘텐츠 확보 + 브랜드 인지도
- **보상**: 매주 N명 추첨, 제품 증정
- **기간**: 4~8주 (장기 운영)
- **해시태그**: #썬데이허그꿀잠
- **채널 포커스**: 인스타그램 중심 + 스레드 보조
- **KPI**: 참여 건수, 도달률, UGC 콘텐츠 수

### 7. 멤버십/회원 이벤트
- **목적**: 기존 고객 재구매 + 신규 회원 가입 유도
- **혜택**: 적립금 2배 / 회원 전용 쿠폰 / 등급별 혜택
- **기간**: 7~14일
- **채널 포커스**: 카카오 알림톡 + 이메일 + 스토어 팝업
- **KPI**: 재구매율, 신규 회원수

### 8. 얼리버드
- **목적**: 다음 시즌 사전 수요 확보
- **할인율**: 10~15% (선착순 한정)
- **기간**: 시즌 말 1~2주
- **채널 포커스**: 스토어 + 카카오 알림톡
- **KPI**: 사전 예약 건수

### 9. 플래시세일
- **목적**: 특정 이벤트(어린이날 등) 당일 최대 전환
- **할인율**: 30~50%
- **기간**: 24시간 이내
- **방식**: 자정 오픈, 재고 소진 시 종료
- **채널 포커스**: 전 채널 동시 알림
- **KPI**: 당일 매출, 완판율

### 10. 신규회원 쿠폰
- **목적**: 첫 방문 → 첫 구매 전환
- **할인율**: 10~15%
- **기간**: 상시 (강화 시기만 지정)
- **채널 포커스**: 스토어 팝업 + 랜딩페이지
- **KPI**: 신규 전환율

## 프로모션 조합 원칙

### 월별 최적 조합

**시즌 시작 월**:
- 메인: 시즌 오픈 세일
- 서브: 체험단 모집 + 신규회원 쿠폰 강화

**이벤트 집중 월**:
- 메인: 번들 세트 특가 or 플래시세일
- 서브: 맘카페 공동구매 + 멤버십 이벤트

**시즌 피크 월**:
- 메인: UGC 캠페인 + 타임특가
- 서브: 체험단 후기 집중 발행 + 얼리버드 예고

### 금지 조합
- 타임특가 + 플래시세일 동시 (할인 혼란)
- 공동구매 + 전체 세일 동시 (공구가 메리트 상실)
- 체험단 모집 + 대규모 할인 동시 (후기 신뢰도 저하)
