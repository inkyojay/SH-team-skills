# 주차별 액션플랜 작성 가이드

## 주차별 액션 구조

각 주차(1W~4W)는 다음 구조로 작성:

```
[주차 번호] [주차 테마 - 한 줄 요약]
├── 스토어: 구체적 실행 항목
├── 네이버: 구체적 실행 항목
├── 인스타: 구체적 실행 항목
├── 스레드: 구체적 실행 항목
├── 맘카페: 구체적 실행 항목
└── 배포형: 구체적 실행 항목 (해당 주에 있을 경우)
```

## 채널별 실행 항목 작성 규칙

### 스토어 (자사몰 + 스마트스토어)
- 배너 교체 내용 명시 (메인 카피 포함)
- 기획전/할인 페이지 설정 내용
- 가격 세팅 (할인율, 쿠폰 등)
- 팝업/알림 설정

### 네이버 (블로그 + 검색)
- 블로그 포스트 제목 예시 (구체적으로)
- 체험단 후기 발행 목표 건수
- 키워드 타겟팅 내용

### 인스타그램
- 콘텐츠 유형 (릴스/피드/스토리/라이브)
- 콘텐츠 주제/카피 예시
- UGC 관련 액션 (리포스트, 당첨자 발표 등)

### 스레드
- 정보형 콘텐츠 주제
- 소통 포맷 (투표, 질문 등)
- 자연스러운 제품 연결 방식

### 맘카페
- 글 유형 (정보글, 추천글, 질문 참여)
- 구체적 글 제목/주제 예시
- 공동구매 관련 액션 (해당 시)

### 배포형 블로그
- 파워블로거 의뢰 건수 + 주제
- 비교 콘텐츠 주제
- 체험단 발행 일정

## 주차 테마 설정 원칙

### 1W (월 첫째 주) — 시작/오픈
- 새로운 캠페인 론칭
- 배너/페이지 세팅
- 기대감 조성 콘텐츠

### 2W (월 둘째 주) — 집중/피크
- 프로모션 최대 강도
- 이벤트 당일 액션
- 전 채널 동시 푸시

### 3W (월 셋째 주) — 확산/바이럴
- 후기 발행 집중
- 맘카페/배포형 집중
- UGC 콘텐츠 수확

### 4W (월 넷째 주) — 정리/예고
- 프로모션 연장 or 마감
- 성과 분석
- 다음 달 티저/예고

## 월간 콘텐츠 수량 기준

| 채널 | 주간 최소 | 월간 목표 |
|------|----------|----------|
| 스토어 배너 | 1회 교체 | 4회 |
| 네이버 브랜드 블로그 | 1건 | 4건 |
| 네이버 체험단 후기 | - | 8~12건 |
| 인스타 릴스 | 1~2건 | 6~8건 |
| 인스타 피드 | 2~3건 | 10~12건 |
| 스레드 | 3~4건 | 12~16건 |
| 맘카페 | 1~2건 | 4~8건 |
| 배포형 | - | 2~4건 |
