# PHASE 7 — 가격 전략 수립 가이드

## 목적

사용자가 제공한 원가 정보 + PHASE 2.5에서 수집한 시장 가격 데이터를 결합하여, 데이터 기반의 가격 전략을 수립한다.

## 사용자에게 수집할 정보

"가격 전략을 세우려면 원가 정보가 필요해요. 알려주실 수 있는 범위에서 공유해주세요:"

| 항목 | 설명 | 필수 여부 |
|------|------|----------|
| 제품 원가 | 제조원가 (자재+인건비+가공비) | 필수 |
| 물류/배송비 | 단위당 배송 비용 | 권장 |
| 목표 마진율 | 원하는 순이익률 (%) | 권장 (없으면 업계 평균 적용) |
| 마케팅 비용 비율 | 매출 대비 마케팅 투자 비율 | 선택 |
| 플랫폼 수수료 | 네이버/쿠팡 수수료 (보통 10~15%) | 권장 |

없는 항목은 업계 평균값으로 추정하되, 반드시 "추정값" 명시.

## 가격 분석 프레임워크

### 1. 원가 구조 분석

```
제품 원가 + 물류비 + 플랫폼 수수료 + 마케팅 비용 = 총 비용
판매가 - 총 비용 = 순이익
순이익 / 판매가 × 100 = 순마진율
```

제품별 손익 시뮬레이션 테이블:
| 제품 | 원가 | 총비용 | 판매가 | 순이익 | 마진율 |
|------|------|--------|--------|--------|--------|

### 2. 시장 가격 포지셔닝 (PHASE 2.5 데이터 연계)

PHASE 2.5 쇼핑 분석에서 나온 가격대 데이터를 활용:
- 시장 평균가 대비 우리 가격의 위치
- 경쟁사 가격대 vs 우리 원가 → 마진 비교
- 가성비 / 주류 / 프리미엄 중 어느 구간에 진입할지

### 3. 가격 전략 유형 선택

| 전략 | 설명 | 적합한 경우 |
|------|------|-----------|
| 프리미엄 가격 | 시장 평균 120~150% | 브랜드 파워 있음 + 품질 차별화 명확 |
| 시장 침투 가격 | 시장 평균 80~95% | 신규 카테고리 진입 + 빠른 점유율 확보 필요 |
| 경쟁 매칭 | 시장 평균과 유사 | 차별화가 기능보다 브랜드/서비스에 있을 때 |
| 번들 프리미엄 | 개별 합계보다 세트가 10~20% 할인 | 객단가 UP + 세트 구매 유도 |
| 심리적 가격 | X9,900원 / X4,900원 형태 | 소비자 가격 저항 최소화 |

### 4. 세트/번들 가격 설계

개별 제품 가격이 정해지면 번들 가격 설계:
- 세트 할인율: 보통 10~20% (합산 대비)
- "세트가 더 이득" 느낌을 주되 개별 제품 마진도 확보
- 앵커링 효과: 세트 원가를 먼저 보여주고 할인가 강조

### 5. 프로모션 가격 시나리오

| 시나리오 | 할인율 | 적용 시점 | 목적 |
|---------|--------|---------|------|
| 얼리버드 | 15~20% | 출시 전 2주 | 초기 판매량 + 리뷰 확보 |
| 런칭 특가 | 10~15% | 출시 후 1개월 | 시장 침투 |
| 정상가 | 0% | 런칭 1개월 후 | 수익 확보 |
| 시즌 세일 | 20~30% | 시즌 말 | 재고 소진 |
| 공동구매 | 15~25% | 리뷰 50개+ 확보 후 | 바이럴 폭발 |

각 시나리오별 예상 매출/이익 시뮬레이션 포함.

### 6. 손익분기점(BEP) 분석

```
고정비 (금형/디자인/초기 마케팅) ÷ (판매가 - 변동비) = BEP 수량
```

시나리오별 BEP:
| 시나리오 | 판매가 | 단위 마진 | 고정비 | BEP 수량 | 예상 달성 시기 |
|---------|--------|---------|--------|---------|-------------|

## 리포트 반영

HTML 리포트의 "가격 전략" 탭에:
- 원가 구조 워터폴 차트 (원가 → 배송 → 수수료 → 마케팅 → 마진)
- 시장 대비 가격 포지셔닝 맵 (경쟁사 가격 vs 우리)
- 프로모션 가격 타임라인
- BEP 시뮬레이션 카드 (보수적/기본/낙관적)
