# 소셜 리스닝 가이드

## 수집 소스별 접근 방법

### 1. 네이버 쇼핑 리뷰
```
URL 패턴: https://search.shopping.naver.com/product/{product_id}/review
수집 항목:
  - 별점 분포
  - 최근 리뷰 텍스트 (최신 20건)
  - 키워드 빈도
주의: 한 번에 과도한 요청 금지, 2~3초 딜레이
```

### 2. 네이버 블로그 (web_search 활용)
```
검색 쿼리 패턴:
  - "{브랜드명} 슬리핑백 후기"
  - "{브랜드명} 사용 후기"
  - "{브랜드명} 리뷰"
수집: 상위 10건 제목 + 요약
감성 판단: 제목/요약의 긍정/부정 키워드 기반
```

### 3. 네이버 카페 (맘카페)
```
검색 대상 카페:
  - 맘카페 (전국 맘들의 모임)
  - 웨딩앤베이비
  - 임신육아 정보 카페
  
검색 쿼리: site:cafe.naver.com "{브랜드명}"
수집: 최근 7일 언급글 제목 + 요약
```

### 4. 인스타그램 댓글
```
접근: 브랜드 최근 게시물 5개의 댓글
감성 키워드 분류로 긍/부/중 판별
```

## 버즈 점수 계산

```python
def calculate_buzz_score(mentions_7d, sentiment_avg, review_count):
    """
    버즈 점수 = (언급량 × 0.4) + (감성 점수 × 0.4) + (리뷰 증가율 × 0.2)
    0~100점 스케일
    """
    volume_score = min(mentions_7d / 50 * 100, 100)  # 50건 = 만점
    sentiment_score = (sentiment_avg + 1) / 2 * 100   # -1~1 → 0~100
    # review_growth는 별도 계산
    
    return volume_score * 0.4 + sentiment_score * 0.4
```

## 맘카페 모니터링 주의사항

- 카페 게시물 전문 수집 금지 (요약만)
- 개인정보 포함 내용 (전화번호, 이름 등) 제외
- 비방성 내용 그대로 저장 금지 → 감성 분류만 저장
- 주 1~2회 이상 과도한 크롤링 금지
