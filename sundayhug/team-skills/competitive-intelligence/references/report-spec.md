# HTML 리포트 스펙 — 경쟁사 인텔리전스

## 리포트 구조 (탭 7개)

### TAB 1: 📊 대시보드 (Overview)
- 조사 일자 + 조사 경쟁사 수
- 금일 주요 변동사항 Highlight 카드 (최대 5건)
  - 🔴 가격 인하 감지
  - 🟡 신규 프로모션
  - 🟢 버즈 급상승
- 경쟁사별 종합 점수 카드 (마케팅 활동지수)

### TAB 2: 🏷️ 경쟁사 맵 (Competitor Map)
- 포지셔닝 매트릭스 (가격대 × 마케팅 강도)
- 경쟁사 카드 그리드 (브랜드명, 로고, 주력제품, 가격대, 채널)
- 직접/간접 경쟁사 구분 뱃지

### TAB 3: 💰 가격/프로모션 (Price & Promo)
- 경쟁사별 현재가 비교 테이블
  - 전일 대비 변동 표시 (↑↓ 화살표 + 변동률)
- 진행 중인 프로모션 타임라인 (간트 형태)
- 세일/이벤트 유형 분포 도넛 차트

### TAB 4: 📱 마케팅 분석 (Marketing Intel)
- 경쟁사별 인스타그램 지표 비교
  - 팔로워 수, 최근 7일 게시물 수, 평균 좋아요
- 콘텐츠 유형 분포 (제품컷/UGC/이벤트/광고)
- 주요 해시태그 Top 20 (버블 차트)
- 체험단/인플루언서 활동 감지 목록

### TAB 5: 👂 소셜 리스닝 (Social Listening)
- 브랜드별 최근 7일 언급량 바 차트
- 감성 분석 결과 (긍정/부정/중립 스택 바)
- 주요 VOC 키워드 워드클라우드 (경쟁사 vs 썬데이허그)
- 맘카페 주요 언급 발췌 (최근 5건, 브랜드별)

### TAB 6: 📅 변동 이력 (Change Log)
- 최근 30일 주요 변동 타임라인
- 날짜 | 경쟁사 | 변동 유형 | 내용 형식
- 필터: 경쟁사 선택 / 변동 유형 선택

### TAB 7: 💡 인사이트 & 액션 (Insights)
- AI 분석 요약 (이번 주 경쟁 환경 변화)
- 썬데이허그 대응 액션 추천 (최대 5건)
  - 우선순위 라벨 (긴급/권장/검토)
- 다음 조사까지 주시 포인트

## 디자인 시스템

```css
/* 브랜드 컬러 */
--primary: #1D9E75;       /* 썬데이허그 그린 */
--primary-light: #E1F5EE;
--danger: #E53E3E;        /* 가격 인하 알림 */
--warning: #EF9F27;       /* 주의 변동 */
--success: #38A169;       /* 긍정 신호 */

/* 경쟁사 구분 컬러 (최대 8개) */
--comp-1: #4299E1;  /* 파랑 */
--comp-2: #ED8936;  /* 오렌지 */
--comp-3: #9F7AEA;  /* 보라 */
--comp-4: #F56565;  /* 빨강 */
--comp-5: #48BB78;  /* 초록 */
--comp-6: #ECC94B;  /* 노랑 */
--comp-7: #76E4F7;  /* 하늘 */
--comp-8: #B794F4;  /* 연보라 */

/* 감성 컬러 */
--sentiment-pos: #48BB78;
--sentiment-neg: #F56565;
--sentiment-neu: #A0AEC0;
```

## 변동사항 뱃지 타입

```
🔴 PRICE_DOWN   — 가격 인하 (즉시 알림)
🟠 PRICE_UP     — 가격 인상
🟡 NEW_PROMO    — 신규 프로모션 시작
🟢 BUZZ_UP      — 버즈 급상승 (+30% 이상)
🔵 NEW_CONTENT  — 신규 마케팅 콘텐츠
⚪ NEW_PRODUCT  — 신제품 출시 감지
```

## 파일 저장 규칙

```
/outputs/ci_report_YYYYMMDD.html    — 일별 리포트
/outputs/ci_weekly_YYYYWW.html      — 주간 요약 리포트
```
