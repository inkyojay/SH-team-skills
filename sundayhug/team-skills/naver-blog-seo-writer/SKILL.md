# 네이버 블로그 SEO 라이터

네이버 상단 노출에 최적화된 블로그 글을 키워드 데이터 기반으로 자동 작성하는 스킬.
네이버 검색광고 API + 네이버 블로그 검색 + Google Gemini 이미지 생성을 연동한다.

---

## 전체 플로우

```
STEP 0  대화형 정보 수집 (주제, 키워드, 브랜드, 톤)
   ↓
STEP 1  키워드 분석 (네이버 API JSON 업로드 or 실시간 입력)
   ↓
STEP 2  상위 블로그 경쟁 분석 (web_search로 상위 글 구조 파악)
   ↓
STEP 3  SEO 최적화 블로그 본문 작성
   ↓
STEP 4  Google Gemini API로 이미지 생성 + 삽입
   ↓
STEP 5  완성본 HTML 파일로 출력 + 발행 가이드
```

---

## STEP 0 — 정보 수집 대화

시작 시 반드시 STEP 0부터 실행. 아래 정보를 자연스럽게 대화로 수집.

**필수 수집 항목:**
| 항목 | 예시 |
|------|------|
| 블로그 주제/제목 방향 | "신생아 슬리핑백 고르는 법" |
| 핵심 타겟 키워드 | "신생아 슬리핑백", "아기 수면자루" |
| 브랜드/제품 | 썬데이허그 슬리핑백, 또는 없음(일반 정보성) |
| 타겟 독자 | 신생아 엄마, 육아 초보, 0-3개월 부모 |
| 글 톤 | 정보성/전문가형, 친근한 엄마 말투, 리뷰형 |
| 키워드 데이터 | API JSON 파일 있음 / 없으면 직접 수집 |
| Google API 키 | 이미지 생성용 (없으면 SKIP) |

**수집 후 확인:** 수집된 정보를 표로 정리하여 사용자에게 확인받고 진행.

## STEP 1 — 키워드 분석

### 1-A. 네이버 API JSON이 있는 경우
사용자가 JSON 파일을 업로드하면 → `references/keyword-analysis-guide.md` 참조하여 분석.

분석 기준:
- **골든 키워드**: 검색량 1,000~10,000 + 경쟁도 낮음 → 메인 타겟
- **서브 키워드**: 검색량 100~1,000 → 소제목/본문에 자연 배치
- **연관 키워드**: LSI 키워드로 본문에 자연스럽게 삽입

### 1-B. API JSON 없는 경우
로컬 스크립트로 수집 안내:

```bash
python scripts/naver_keyword_search.py \
  --api-key YOUR_API_KEY \
  --secret-key YOUR_SECRET_KEY \
  --customer-id YOUR_CUSTOMER_ID \
  --keywords "신생아 슬리핑백,아기 수면자루,슬리핑백 추천,신생아 이불" \
  --output keyword_results.json
```

결과 JSON을 대화창에 업로드해달라고 요청.

### 1-C. API 없는 경우 (폴백)
web_search로 블랙키위/SURF 기준 키워드 트렌드 추정 + 사용자 제공 정보로 진행.

---

## STEP 2 — 상위 블로그 경쟁 분석

`web_search`로 타겟 키워드 상위 노출 블로그 3~5개 분석.

**분석 항목:**
1. 제목 패턴 (숫자형/질문형/추천형 등)
2. 글 구조 (소제목 수, 순서, 패턴)
3. 평균 글 길이 추정
4. 이미지 사용 방식
5. 키워드 배치 위치 (제목/첫 문단/소제목/마지막)

**분석 후 결론 도출:**
- 상위 글들의 공통 구조 패턴
- 우리 글이 차별화할 포인트
- 추가로 다뤄야 할 정보성 요소

---

## STEP 3 — SEO 최적화 블로그 본문 작성

`references/naver-seo-rules.md`와 `references/blog-structure.md`를 반드시 참조 후 작성.

### 핵심 SEO 규칙 요약

**제목:** 
- 메인 키워드를 제목 앞 1/3 내에 배치
- 클릭을 유도하는 숫자/감성 단어 포함
- 35자 이내 권장
- 예: `신생아 슬리핑백 고르는 법 5가지 | 육아 전문가가 알려주는 체크리스트`

**본문 구조:**
```
[도입부] (200-300자)
 → 공감/문제 제기 → 메인 키워드 첫 등장 → 글에서 얻을 것 예고

[소제목 1] H2
 → 핵심 내용 1 (메인 키워드 재등장)
 → 서브 키워드 자연 포함
 → [이미지 1 삽입 위치]

[소제목 2] H2
 → 핵심 내용 2
 → [이미지 2 삽입 위치]

... (소제목 4~6개)

[마무리/CTA] (100-200자)
 → 요약 → 행동 유도 → 브랜드 언급 (있으면)
```

**키워드 배치:**
- 제목: 메인 키워드 1개
- 첫 문단 100자 내: 메인 키워드
- 소제목(H2): 메인 or 서브 키워드
- 본문 전체: 키워드 밀도 1.5~2.5%
- 마지막 문단: 메인 키워드 재등장

**글 분량:** 최소 1,800자, 권장 2,500~3,500자

## STEP 4 — Google Gemini 이미지 생성

### 이미지 생성 방식

사용자가 Google API 키를 제공한 경우 → `scripts/google_imagen.py` 실행.
제공하지 않은 경우 → 이미지 프롬프트만 제공하고 SKIP.

**이미지 삽입 기준:**
- 블로그 글 내 3~5개 이미지 권장
- 도입부 후 1개 (대표 이미지)
- 소제목 1~2개마다 1개
- 마지막 정리 전 1개

**이미지 생성 실행:**
```bash
python scripts/google_imagen.py \
  --api-key YOUR_GOOGLE_API_KEY \
  --prompts "prompt1.json" \
  --output-dir blog_images/
```

**프롬프트 생성 원칙:** `references/image-prompt-guide.md` 참조.

---

## STEP 5 — 완성본 출력

### 출력 형식

HTML 파일로 완성본 생성:
- 네이버 블로그 에디터에 바로 복사 가능한 형식
- 이미지 삽입 위치 명확히 표시
- 발행 전 체크리스트 포함
- 키워드 SEO 점수 요약

### 파일 저장
```
/mnt/user-data/outputs/
├── blog_[주제]_[날짜].html    ← 메인 완성본
├── blog_[주제]_[날짜].md     ← 마크다운 버전
└── blog_images/              ← 생성된 이미지들
    ├── image_01.png
    ├── image_02.png
    └── ...
```

## 주의사항

- **브랜드 글**: 정보성 80% + 브랜드 20% 비율 유지. 광고성 글로 느껴지면 C-Rank 하락.
- **복붙 금지**: 경쟁 글 구조는 참고만, 문장 절대 복사 금지.
- **이미지 alt 텍스트**: 반드시 키워드 포함 설명 제공.
- **발행 시간**: 평일 오전 7~9시, 오후 7~9시 권장.
- **해시태그**: 글 말미 관련 태그 10~15개 자동 생성.
