# 인스타그램 디자인 템플릿 시스템

> 썬데이허그 브랜드의 인스타그램 디자인 템플릿
> HTML → 이미지 변환 용도 · JS/애니메이션 없음

## 개요

| 항목 | 내용 |
|------|------|
| 콘텐츠 타입 | 6종 |
| 디자인 베리에이션 | 18개 |
| 캠페인 팔레트 | 12개 |
| 변환 방식 | HTML → PNG (브라우저 캡처) |

## 파일 구조

```
instagram/
├── _base-styles.css          ← 인스타그램 전용 CSS (.ig- 접두사)
├── _palettes.css             ← 12개 캠페인 팔레트
├── _guide-chrome.css         ← 가이드 UI 스타일
├── GUIDE.md                  ← 이 문서
├── template-guide.html       ← 비주얼 가이드 (타입 탭 + 팔레트 전환)
└── types/                    ← 콘텐츠 타입별 HTML 템플릿
    ├── 01-single-feed/       ← 싱글 피드 (1080×1080)
    ├── 02-card-news/         ← 카드뉴스 (1080×1350, 멀티 슬라이드)
    ├── 03-reels-cover/       ← 릴스 커버 (1080×1920)
    ├── 04-story/             ← 스토리 (1080×1920)
    ├── 05-ad-image/          ← 광고 이미지 (1080×1080)
    └── 06-highlight-cover/   ← 하이라이트 커버 (1080×1080)
```

| # | 타입명 | 이미지 크기 | 비율 | 베리에이션 | 용도 |
|---|--------|------------|------|-----------|------|

| 06 | 하이라이트 커버 | 1080×1080 | 1:1 | A-아이콘 / B-텍스트 / C-미니제품 | 프로필 하이라이트 커버 |

## 베리에이션 전략

- **A**: 제품 중심 — 제품 이미지 + 가격/할인 정보 강조
- **B**: 감성/스토리 — 라이프스타일 이미지, 리뷰, 혜택 정보
- **C**: 이벤트/특수 — 이벤트, 비교, 투표, 사회적 증거

### 카드뉴스 슬라이드 구성

카드뉴스(02)는 **3장 슬라이드**로 구성:
- **슬라이드 1**: 커버 (제목 + 브랜드)
- **슬라이드 2**: 본문 콘텐츠 (정보/스토리/비교)
- **슬라이드 3**: CTA (제품 이미지 + 구매 유도)

## 사용법

### 1. 템플릿 선택

목적에 맞는 타입과 베리에이션을 선택합니다.

### 2. 이미지 교체

`src="..."` 경로를 실제 제품 이미지로 교체합니다.

### 3. 텍스트 수정

제품명, 가격, 이벤트 정보 등을 실제 콘텐츠로 교체합니다.

### 4. 팔레트 적용

`<body>` 태그에 `data-palette` 속성을 추가합니다.

```html
<body data-palette="warm-spring" style="margin:0;padding:0;background:#eee;">
```

### 5. 이미지 변환

HTML 파일을 브라우저에서 열고, 해당 프레임을 캡처합니다.
- 정확한 픽셀 크기로 캡처 (1080×1080, 1080×1350, 1080×1920)
- `object-fit: cover`로 이미지 비율 자동 조정

## 제약사항

- **JS/애니메이션 완전 금지** — HTML → 이미지 변환 용도
- **인스타그램 규격 준수** — 1080×1080, 1080×1350, 1080×1920
- **`object-fit: cover`** 필수 — 이미지 비율 강제
- **midnight-luxe 팔레트** — 다크 배경 별도 오버라이드 적용됨

## 검증 체크리스트

1. `template-guide.html`을 브라우저에서 열기 (file:// 호환)
2. 6개 타입 탭 전환 확인
3. 12개 팔레트 전환 시 모든 베리에이션에 색상 적용 확인
4. 각 프레임이 정확한 인스타그램 규격 크기인지 개발자도구로 확인
5. 이미지 렌더링 시 잘림/여백 없는지 확인
