# 카피 생성 프레임워크

## 톤별 카피 가이드

### 1. 감성 (Emotional)

**목적**: 공감, 감정 자극, 라이프스타일 연결
**어조**: 따뜻한, 부드러운, 공감형

**헤드라인 패턴:**
- "{감정/상황} {솔루션}"
  - 예: "아기의 꿀잠, 엄마의 편안함"
- "{타겟}을 위한 {베네핏}"
  - 예: "우리 아이 첫 겨울을 위한 포근함"
- 질문형 공감
  - 예: "밤마다 이불 걷어차는 우리 아이, 걱정되시죠?"

**Primary Text 패턴:**
```
{공감 문장}
{제품이 해결하는 방법}
{구체적 베네핏 1~2개}
{CTA}
```

**예시:**
```
아이가 밤새 편하게 자는 모습, 상상만 해도 행복하죠 😊
썬데이허그 슬리핑백은 입는 이불이라 걷어차도 괜찮아요.
✨ 새벽 이불 덮어주기 끝
지금 만나보세요 →
```

---

### 2. 정보형 (Informational)

**목적**: 스펙, 기능, 비교 우위 전달
**어조**: 신뢰감 있는, 전문적, 명확한

**헤드라인 패턴:**
- "{숫자} {키워드}"
  - 예: "4중 거즈로 만든 사계절 슬리핑백"
- "{기능} + {결과}"
  - 예: "양면 사용 가능, 1개로 2개 효과"
- 비교형
  - 예: "일반 이불 vs 슬리핑백, 뒤척임 비교"

**Primary Text 패턴:**
```
{핵심 기능/스펙 강조}
{구체적 수치/비교}
{사용 시나리오}
{CTA}
```

**예시:**
```
🔬 국내 KC인증 완료 | OEKO-TEX 100 인증 원단
대나무 원단으로 일반 면 대비 통기성 3배
여름에도 쾌적하게, 겨울에도 따뜻하게 — 사계절 OK
상세 스펙 확인하기 →
```

---

### 3. 긴급성 (Urgency)

**목적**: 즉각 행동 유도, FOMO 자극
**어조**: 강렬한, 직접적, 시간 압박

**헤드라인 패턴:**
- "[기간] {혜택}"
  - 예: "오늘만! 슬리핑백 30% 할인"
- 한정형
  - 예: "선착순 100명 무료배송"
- 숫자 강조
  - 예: "29,900원 → 19,900원"

**Primary Text 패턴:**
```
🔥 {긴급 상황/한정 정보}
{혜택 상세}
{마감 시점 강조}
{즉각 CTA}
```

**예시:**
```
🔥 봄맞이 특가 D-3
슬리핑백 전 사이즈 30% OFF
추가 5,000원 할인쿠폰까지!
⏰ 일요일 자정 마감 — 지금 바로 →
```

## CTA 가이드

| CTA | 용도 | 톤 매칭 |
|-----|------|---------|
| 지금 만나보기 | 브랜드 인지도, 감성 | Emotional |
| 자세히 보기 | 정보 탐색, 스펙 확인 | Informational |
| 지금 구매하기 | 직접 전환 | Urgency |
| 할인 받기 | 프로모션 | Urgency |
| 무료배송으로 받기 | 혜택 강조 | Informational |
| 리뷰 확인하기 | 사회적 증거 | Informational |

## 카피 길이 가이드 (이미지 내 텍스트)

| 사이즈 | 헤드라인 | 서브텍스트 | 
|--------|---------|-----------|
| 1080×1080 | 8~15자 | 15~25자 |
| 1080×1350 | 10~18자 | 20~30자 |
| 1080×1920 | 12~20자 | 25~40자 |

이미지 내 텍스트는 짧을수록 좋다. 핵심 메시지만 넣고 상세 내용은 Primary Text에.

## USP → 카피 변환 예시

**USP**: "대나무 원단으로 통기성이 뛰어남"

| 톤 | 헤드라인 | Primary Text |
|----|---------|-------------|
| 감성 | 시원한 잠자리, 행복한 아침 | 여름밤 뒤척이는 우리 아이를 위해 🌿 대나무 원단의 시원한 감촉으로 꿀잠을 선물하세요 |
| 정보형 | 대나무 원단, 통기성 3배 | 🔬 대나무(뱀부) 원단은 일반 면 대비 통기성 3배. 여름에도 쾌적한 수면 환경을 만들어줍니다 |
| 긴급 | 여름 준비 끝! 한정특가 | 🔥 무더위 전 준비하세요! 대나무 슬리핑백 여름 한정 30% 할인. 이번 주까지만! |
