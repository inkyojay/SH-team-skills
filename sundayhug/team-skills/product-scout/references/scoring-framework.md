# 기회 스코어링 프레임워크

## 스코어 공식

```
기회 점수 (0~100) = 
  검색량 점수 (0~30) + 
  경쟁 점수 (0~25) + 
  마진 점수 (0~25) + 
  시너지 점수 (0~20)
```

## 요소별 계산

### 1. 검색량 증가율 점수 (0~30)

네이버 데이터랩 주간 변화율 기준:

| 증가율 | 점수 |
|--------|------|
| 50%+ | 30 |
| 30~49% | 25 |
| 20~29% | 20 |
| 10~19% | 15 |
| 5~9% | 10 |
| 0~4% | 5 |
| 감소 | 0 |

보정 요소:
- 절대 검색량 1만+ 이면 +5 (최대 30)
- 3개월 연속 증가 시 +5 (최대 30)
- 시즌 키워드면 시즌 진입 시점에 +5

### 2. 경쟁 강도 점수 (0~25)

경쟁도 = 검색광고 경쟁도 + 네이버 쇼핑 등록 상품 수

| 경쟁도 | 등록 상품 수 | 점수 |
|--------|-------------|------|
| 낮음 | < 100 | 25 |
| 낮음 | 100~500 | 20 |
| 중간 | < 100 | 20 |
| 중간 | 100~500 | 15 |
| 중간 | 500+ | 10 |
| 높음 | < 500 | 10 |
| 높음 | 500+ | 5 |

보정 요소:
- 국내 브랜드 0개 (직구만 존재) → +5 (최대 25)
- 전문 브랜드 1~2개만 → +3

### 3. 마진 가능성 점수 (0~25)

마진율 = (예상 소비자가 - 예상 원가) / 예상 소비자가

| 예상 마진율 | 점수 |
|------------|------|
| 70%+ | 25 |
| 60~69% | 22 |
| 50~59% | 18 |
| 40~49% | 14 |
| 30~39% | 10 |
| 30% 미만 | 5 |

예상 원가 산출:
- 1688 검색 평균가 × 1.3 (관세/물류)
- 예상 소비자가: 유사 국내 제품 평균가 또는 해외 직구가 × 0.7

### 4. 라인업 시너지 점수 (0~20)

| 시너지 수준 | 점수 | 예시 |
|------------|------|------|
| 직접 확장 | 20 | 슬리핑백 → 여름 슬리핑백 |
| 인접 카테고리 | 15 | 슬리핑백 → 수면조끼 |
| 교차판매 가능 | 10 | 슬리핑백 → 베이비 모니터 |
| 브랜드 적합 | 5 | 수면 용품 → 아기 가구 |
| 무관 | 0 | 수면 용품 → 이유식 |

## 등급 기준

| 점수 | 등급 | 액션 |
|------|------|------|
| 80~100 | A (즉시 검토) | new-product-planner 연결, 소싱 시작 |
| 60~79 | B (추가 조사) | competitive-intelligence로 심층 분석 |
| 40~59 | C (모니터링) | 다음 주 재평가 |
| 0~39 | D (보류) | 분기 재검토 |

## 시그널 가중치 보정

여러 시그널이 동시에 감지되면 추가 보정:

| 동시 시그널 | 보정 |
|------------|------|
| 검색량↑ + 맘카페 VOC | +5 |
| 해외 트렌드 + 국내 경쟁 0 | +5 |
| 검색량↑ + 해외 트렌드 + VOC | +10 |
| 3개 이상 시그널 동시 감지 | +10 (최대) |

최종 점수는 100점을 초과하지 않는다.
