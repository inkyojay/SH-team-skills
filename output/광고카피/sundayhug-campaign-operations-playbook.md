### 소재(Ad) 레벨 공통 설정

| 항목 | 설정 |
|------|------|
| Primary Text | 3~5개 변형 등록 (Advantage+ Creative) |
| Headline | 2~3개 변형 등록 |
| Description | 1~2개 |
| CTA 버튼 | Shop Now |
| URL | sundayhug.kr/제품페이지 (제품별 다이렉트 링크) |
| URL Parameters | utm_source=meta&utm_medium=paid&utm_campaign={campaign_name}&utm_content={ad_name} |
| Advantage+ Creative | ON (자동 크롭, 밝기 조정 등 허용) |

### Phase 1: 런칭 + 데이터 축적 (D+0 ~ D+6)

> **이 기간의 철칙: 아무것도 건드리지 않는다.**

| 날짜 | 상태 | 예상 데이터 | 액션 |
|------|------|------------|------|
| **D+0 (런칭일)** | 광고 게재 시작 | 노출 시작, 전환 없을 수 있음 | 게재 승인 확인만. **수정 금지** |
| **D+1** | 학습 중 | CPM 높을 수 있음 (15,000~30,000원) | 패닉 금지. 정상임 |
| **D+2** | 학습 중 | 클릭 데이터 축적 시작 | CTR 확인만 (1% 미만이어도 정상) |
| **D+3** | 초기 데이터 | 첫 전환 발생 가능 | ROAS 판단하지 않기 (데이터 부족) |
| **D+4** | 데이터 축적 | CPA 윤곽 잡히기 시작 | 관찰만 |
| **D+5** | 데이터 축적 | Ad Set별 지출 편차 발생 | 관찰만. CBO가 알아서 배분 중 |
| **D+6** | 분석 준비 | 7일치 데이터 거의 완성 | 내일 분석 준비 |

**D+0 ~ D+6 기간 절대 하지 말 것:**
- Ad Set ON/OFF
- 예산 변경
- 타겟 변경
- 소재 추가/삭제
- 비딩 전략 변경

### Phase 4: 두 번째 분석 (D+14)

> **Phase 1 런칭분 2주차 + Phase 3 이터레이션 1주차 동시 분석**

#### D+14 판단 매트릭스 (강화 버전)

| 상황 | 조건 | 액션 |
|------|------|------|
| **위너 확정** | 2주 연속 ROAS 3.0x+, 자연 지출 증가 | **Core Concept으로 등록**. 이터레이션 대량 생산 체제 돌입 |
| **위너 → 하락** | 1주차 위너가 2주차 ROAS 하락 | 소재 피로도 가능. 이터레이션으로 대체 |
| **유망 → 위너 승격** | 관찰 중이던 세트가 ROAS 3.0x+ 달성 | Minimum 유지/제거 후 이터레이션 기획 |
| **유망 → 루저** | 2주 관찰 후에도 ROAS 2.0x 미달 | 해당 컨셉 종료. 새 컨셉으로 교체 |
| **이터레이션 성공** | 새 이터레이션 ROAS가 원본 이상 | 해당 변형을 새 Core로 추가 |
| **이터레이션 부진** | 새 이터레이션 ROAS가 원본 미만 | 변형 방향 재검토. 다른 요소 변경 시도 |

## 4. 예산 스케일링 로드맵

### 예산 증액 판단 기준

```
조건 충족 시에만 증액:
✅ 최근 7일 ROAS ≥ 목표 ROAS (3.0x)
✅ CPA가 안정적 (급등 없음)
✅ Core Concept 1개 이상 확보
✅ 전환 볼륨이 일 5건 이상
```

### 스케일링 타임라인

| 시점 | 일 예산 | 월 예산 | 조건 |
|------|---------|---------|------|
| **D+0 (시작)** | 200,000원 | 6,000,000원 | - |
| **D+14~D+21** | 250,000원 | 7,500,000원 | ROAS 3.0x+ 2주 유지 시 |
| **D+28~D+35** | 300,000원 | 9,000,000원 | ROAS 유지 + Core Concept 2개+ |
| **Month 2 중반** | 400,000원 | 12,000,000원 | ROAS 유지 + Core Concept 3개+ |
| **Month 3** | 500,000원+ | 15,000,000원+ | 플라이휠 안정. M3 Cost Cap 도입 검토 |

### 증액 방법

```
❌ 잘못된 방법: 캠페인 예산 200,000 → 300,000 한번에 50% 증액
✅ 올바른 방법:

방법 1 (권장): 새 Ad Set 추가로 자연 증액
  - 기존 Ad Set 그대로 + 새 Ad Set(이터레이션) 런칭
  - CBO가 자연스럽게 총 지출 증가
  - 캠페인 예산을 새 Ad Set 수에 맞게 조정

방법 2: 캠페인 예산 단계 증액
  - 140,000 → 170,000 → 200,000 (20% 단위)
  - 각 증액 후 3~5일 안정화 대기
```

## 6. 트러블슈팅 가이드

### 상황별 즉시 대응

| 상황 | 원인 가능성 | 즉시 액션 |
|------|------------|----------|
| **전환 0건 (3일 이상)** | Pixel 이벤트 미발화 | Events Manager에서 Purchase 이벤트 확인 |
| **CTR 0.5% 미만** | 소재가 타겟에 안 맞음 | 소재 교체 준비 (7일 기다린 후 실행) |
| **CPM 30,000원 이상** | 타겟 너무 좁음 or 시장 경쟁 | 타겟 확장 (연령대 넓히기) |
| **ROAS 갑자기 50% 하락** | 소재 피로 or 시장 변화 | 새 이터레이션 긴급 제작. 1주 관찰 |
| **특정 Ad Set만 지출 독점** | CBO 최적화 편향 | 다른 Ad Set에 Minimum 재설정 |
| **Retargeting ROAS 1.0x 미만** | 오디언스 풀 너무 작음 | Prospecting 볼륨 먼저 키우기 |

## 7. 핵심 원칙 요약 (벽에 붙여놓기)

```
1. 7일 안에 건드리지 마라
2. 데이터로 판단하라 (감으로 하지 마라)

6. Incremental Attribution으로 분석하라
7. 전환 높은 요일에 더 써라
8. Prospecting에 70% 이상 투자하라
9. 수익성 있으면 ROAS 낮아도 운영하라
10. 매주 월요일 분석, 금요일 런칭
```
