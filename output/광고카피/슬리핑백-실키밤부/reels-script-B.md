## 제작 가이드

### 편집 스타일
- **총 길이**: 25초 (빠른 템포)
- **컷 수**: 7-9컷
- **페이스**: 비트에 맞춘 빠른 편집 (2-4초/컷)
- **핵심**: 각 클립마다 리뷰 텍스트 → "많은 엄마가 인정" 느낌

```css
  background: rgba(255, 255, 255, 0.85);

}

  color: #FFD700;
  font-size: 24px;
}
.review-text {
  color: #3E2723;
  font-family: 'Pretendard';
  font-weight: 500;
  font-size: 32px;
  line-height: 1.4;
}
```

### 자막 스타일
- **폰트**: Pretendard (전 장면 통일)
- **리뷰 카드**: 반투명 화이트 배경 + 다크 브라운 텍스트
- **일반 텍스트**: 흰색 + 드롭 섀도우
- **위치**: 반드시 세이프존(900x1492px) 내

- **장르**: 밝은 어쿠스틱 팝 / 하우스 라이트

- **포인트**: 각 클립 전환이 비트에 정확히 맞도록
